Сумма цифр

Напишите программу, которая считывает трехзначное положительное число (u16) и выводит сумму его цифр.

Чтобы найти сумму цифр трехзначного числа, можно разложить число на отдельные цифры и затем сложить их.

Тестовые данные ✅

Sample Input:
123

Sample Output:
6

Напишите программу. Тестируется через stdin → stdout

use std::io;
use std::str::FromStr;
use std::fmt::Debug;
fn read_number<T: FromStr>() -> T where <T as FromStr>::Err: Debug {
    let mut user_input = String::new();
    io::stdin().read_line(&mut user_input).expect("Не удалось прочитать ввод");
    user_input.trim().parse().expect("Не удалось преобразовать в число")
}

fn main() {
    let i: i32 = read_number(); 
    println!("{}", i / 100 + i % 100 / 10 + i % 10);   
}
