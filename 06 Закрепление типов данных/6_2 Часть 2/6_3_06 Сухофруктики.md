Сухофруктики

Из некоторой массы свежих фруктов при сушке получили какую-то массу сушеных. Напишите программу, которая считывает два вещественных числа: массу исходного сырья в кг и получившуюся массу сухофруктов. Программа должна выводить в процентах (до 3 знаков) какую часть от исходной массы фруктов составляет масса сухофруктов и сколько процентов массы теряется при сушке, как показано в примере.

Гарантируется, что исходная масса больше 0 и масса сухофруктов не превышает исходную.

Тестовые данные ✅

Sample Input:
1000
100

Sample Output:
Доля сухофруктов относительно свежих фруктов составляет: 10.000%
Процент массы, потерянный при сушке: 90.000%

Напишите программу. Тестируется через stdin → stdout
Верно решили 117 учащихся
Из всех попыток 63% верных


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {
    let f1: f64 = input();   
    let f2: f64 = input();     
    println!("Доля сухофруктов относительно свежих фруктов составляет: {:.3}%\n\
              Процент массы, потерянный при сушке: {:.3}%", (f2 * 100.0) / f1, 100.0 - (f2 * 100.0) / f1);
}
