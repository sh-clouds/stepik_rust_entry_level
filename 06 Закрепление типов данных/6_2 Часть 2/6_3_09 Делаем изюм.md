Делаем изюм

Изюм получается в процессе сушки винограда. Напишите программу, которая считывает три вещественных числа: необходимое количество изюма в килограммах, количество воды винограда (в %) и количество воды изюма (в %), а затем выводит в килограммах (до 3 знаков) сколько килограммов винограда потребуется для получения нужной массы изюма, как показано в примере.

Тестовые данные ✅
Sample Input:
25
85
5

Sample Output:
Для получения 25.000 кг изюма необходимо 158.333 кг винограда


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {
    let f1: f64 = input(); // необходимое количество изюма в килограммах    
    let f2: f64 = input(); // количество воды винограда (в %) 
    let f3: f64 = input(); // количество воды изюма (в %)         
    println!("Для получения {:.3} кг изюма необходимо {:.3} кг винограда", f1, f1 * (100.0-f3) / (100.0-f2) );
}
