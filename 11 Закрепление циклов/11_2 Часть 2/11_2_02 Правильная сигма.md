Правильная сигма ∑

Напишите программу, которая считывает натуральное число n (u32) и вещественное число x, а затем выводит (до 2 знаков) значение S, вычисленное по формуле:

S=∑_i=0_n x/(2x+1)


Гарантируется, что x≠−1/2.

Тестовые данные ✅

Sample Input:
1
1

Sample Output:
0.67

Напишите программу. Тестируется через stdin → stdout
Верно решили 32 учащихся
Из всех попыток 73% верных



use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {
    let n: i32 = input();
    let x: f64 = input();
    let mut s: f64 = 0.0;
    
    for _ in 0..=n 
    {    
        s += x / (2.0 * x + 1.0); 
    }  
    
    println!("{s:.2}");
}
