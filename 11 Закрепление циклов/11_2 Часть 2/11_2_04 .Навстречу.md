Навстречу  ln2


Напишите программу, которая считывает натуральное число n (usize) и выводит (до n знаков) сумму ряда Маклорена для ln2:

ln2 = 1 - 1/2 +1/3 - 1/4  + ... + 1/n
 

Тестовые данные ✅

Sample Input 1:
1

Sample Output 1:
1.0


Sample Input 2:
2

Sample Output 2:
0.50


Напишите программу. Тестируется через stdin → stdout
Верно решили 30 учащихся
Из всех попыток 81% верных



use std::{io::stdin, str::FromStr};

fn input<T: FromStr>() -> Result<T, T::Err> {
    let mut in_str = String::new();
    stdin().read_line(&mut in_str).unwrap();
    in_str.trim().parse()
}

fn main() {
    let mut n: usize = input().unwrap();
    let mut res: f64 = 0.0;

    for i in 1..=n {
        let seq = 1.0 / (i as f64);
        if i & 1 == 1 { res += seq } else { res -= seq };
    };

    println!("{res:.n$}");
}
