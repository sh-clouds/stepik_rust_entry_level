Игральная кость

При двукратном бросании игральной кости в сумме выпало N очков. Напишите программу, которая считывает натуральные (u8) числа N и M, а затем выводит (до 2 знаков) вероятность того, что хотя бы раз выпало M очков.

Гарантируется, что N > M!

Тестовые данные ✅

Sample Input:
2
1

Sample Output:
1.00

Напишите программу. Тестируется через stdin → stdout

fn probability_at_least_one_m(target_sum: i32, specific_value: i32) -> f64 {
    let mut total_outcomes = 0;
    let mut favorable_outcomes = 0;

    // Перебираем все возможные пары (x, y) для двух бросков
    for first_throw in 1..=6 {  // Первый бросок
        for second_throw in 1..=6 {  // Второй бросок
            if first_throw + second_throw == target_sum {  // Проверяем, что сумма равна N
                total_outcomes += 1;
                if first_throw == specific_value || second_throw == specific_value {  // Проверяем, выпало ли M
                    favorable_outcomes += 1;
                }
            }
        }
    }

    // Вероятность того, что хотя бы раз выпало M очков
    let probability = if total_outcomes > 0 {
        favorable_outcomes as f64 / total_outcomes as f64
    } else {
        0.0
    };

    probability
}

// Считываем значения N и M
use std::io;

fn main() {
    let mut input = String::new();
    io::stdin().read_line(&mut input).unwrap();
    let target_sum = input.trim().parse::<i32>().unwrap();

    input.clear();
    io::stdin().read_line(&mut input).unwrap();
    let specific_value = input.trim().parse::<i32>().unwrap();

    // Проверяем корректность ввода
    if target_sum <= specific_value {
        println!("Ошибка: N должно быть больше M.");
    } else {
        let result = probability_at_least_one_m(target_sum, specific_value);
        println!("{:.2}", result);
    }
}
