Циклы

Вот мы и подошли к ещё одной фундаментальной теме программирования. Большая часть уже пройдена, с чем мы вас и поздравляем.

Часто бывает полезно, а иногда необходимо выполнить блок кода более одного раза. Например, вам понадобилось посчитать сумму всех чисел от 1 до 100. Можно, конечно, создать сто строк кода:

fn main() {
    let mut sum = 0;
    sum += 1;
    sum += 2;
    sum += 3;
    //.......
    sum += 100;
    println!("{sum}");
}
Или другой пример. Увеличить все элементы массива на какое-то число:

fn main() {
    let mut sum = [0; 100];
    sum[0] += 1;
    sum[1] += 1;
    sum[2] += 1;
    //..........
    sum[99] += 1;
    println!("{:?}",sum);
}
Согласитесь, немного утомительно, самую малость 😄. Тут-то нам и приходят на помощь циклы. Цикл (loop) - это управляющая конструкция высокоуровневого языка программирования для многократного исполнения блока кода.

Rust предоставляет три вида циклов:for, while и loop. Начнём с for.



Цикл for

for является наиболее часто используемой конструкцией цикла в Rust и применяется для:

Выполнения блока кода указанное количество раз, т.е. когда известно необходимое количество повторений (итераций);
Перебора коллекций (например, массивы).

Итерация - единичное повторение какого-либо цикла.
 
Цикл создаётся с помощью ключевого слова for, затем указывается название переменной цикла в данном случае i, далее после in необходимый целочисленный диапазон (range) start..end и внутри фигурных скобок {} помещается блок кода, именуемый телом цикла:

for переменная_цикла in диапазон {
    // тело
}
Начнём с простенького примера. Выведем строку Rust 10 раз:

fn main() {
    for i in 0..10 {
        println!("Rust");
    }
}

Игровая площадка ⛱️

Выражение диапазона 0..10 генерирует последовательность всех целых чисел, начиная с 0 и заканчивая 10, но не включая его, заставляя цикл выполняться 10 раз (более подробно рассмотрим в 9 шаге). Далее внутри {} заключается выполняемый код (тело) т.е. println!("Rust");.

Важно, чтобы начало start и конец end удовлетворяли условию start < end. 


Для включения конца диапазона необходимо поставить перед ним знак =:

fn main() {
    for i in 0..=10 {
        println!("Rust");
    }
}
Игровая площадка ⛱️

В более общем случае цикл for выглядит как:

for переменная_цикла in выражение {
    // тело
}

В данном случае выражение - это элемент, который можно преобразовать в итератор. Итератор возвращает последовательность (серию) элементов по одному элементу на итерацию цикла.

Забегая вперед, диапазоны представлены структурами Range и RangeInclusive в стандартной библиотеке Rust.

Range - представляет полуоткрытый диапазон без включения конечного значения 0..10.
RangeInclusive - представляет закрытый диапазон, включая начальное и конечное значения 0..=10.

Обе структуры реализуют типаж (trait) Iterator, позволяющий перебирать элементы в диапазоне.



Переменный диапазон

Начало и конец диапазона могут быть заданы переменными, что позволяет нам манипулировать количеством итераций цикла:

fn main() {
    let (start, end) = (1, 10);

    for i in start..=end {
        println!("Rust");
    }
}

Игровая площадка ⛱️

Тип чисел диапазона, как и для целых чисел, по умолчанию равен i32. Но если вам требуется другой тип, воспользуйтесь суффиксом типа, указав его после start или end:

fn main() {
    for i in 0..10i8 {
        println!("Rust");
    }
}
