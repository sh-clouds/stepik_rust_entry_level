Последовательность Фибоначчи

Напишите программу, которая считывает количество элементов последовательности n (u8) и выводит последовательность Фибоначчи для числа n, как показано в примерах.

Числа Фибоначчи - последовательность чисел, в которой каждое последующее число равно сумме двух предыдущих чисел.

use std::{fmt::Debug, io::stdin, str::FromStr};

fn read_value<T: FromStr>() -> T
where
    <T as FromStr>::Err: Debug, {
    let mut input = String::new();
    stdin().read_line(&mut input).unwrap();
    input.trim().parse::<T>().unwrap()
}

fn main() {
    let mut num: u32 = read_value();
    let mut prev = 0_u32;
    let mut next = 1_u32;
    
    println!("Последовательность Фибоначчи для {num} элементов:\n0");
    if num > 1 {
        println!("1");
    }
    for i in 2..num {
        println!("{}", prev + next);
        (prev, next) = (next, prev + next);
    }
}
