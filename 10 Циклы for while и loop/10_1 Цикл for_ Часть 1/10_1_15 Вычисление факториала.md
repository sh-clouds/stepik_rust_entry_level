Вычисление факториала

Напишите программу, которая считывает целое неотрицательное число n (u32) и выводит для него факториал в виде сообщения:

Факториал числа {n} равен: {}

Факториал натурального числа n определяется, как произведение всех натурального чисел от 1 до n включительно. Для n = 0 принимается в качестве соглашения, что 0!=1.

Тестовые данные ✅

Sample Input 1:
0

Sample Output 1:
Факториал числа 0 равен: 1


Sample Input 2:
1

Sample Output 2:
Факториал числа 1 равен: 1


Sample Input 3:
2

Sample Output 3:
Факториал числа 2 равен: 2

Напишите программу. Тестируется через stdin → stdout

use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {            
    let mut n: i32 = input();  
    let mut fact: i32 = 1; 

    if n == 0
    {
        println!("Факториал числа {n} равен: 1");     
    }
    else
    {
        for i in 1..=n {
            fact *= i
        }
        println!("Факториал числа {n} равен: {fact}");     
    }    
}
