Только вверх

Напишите программу, которая считывает многозначное целое число (u32) и выводит, расположены ли его цифры в порядке возрастания.

Если цифры расположены по возрастанию, вывести Цифры числа {} расположены по возрастанию;
Иначе вывести Цифры числа {} не расположены по возрастанию.
Тестовые данные ✅

Sample Input:
123

Sample Output:
Цифры числа 123 расположены по возрастанию


Напишите программу. Тестируется через stdin → stdout
Верно решили 53 учащихся
Из всех попыток 61% верных



use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {
    let n: u32 = input();
    let s1 = n.to_string();  
    let s2 = &s1[1..]; 
    let mut f = true;
    let mut ch0 = s1.chars().nth(0).unwrap();
    
    for ch in s2.chars() {
        if ch <= ch0
        {
            f = false;
            break;
        }
        ch0 = ch;
    } 
    
    if f == true
    {
        println!("Цифры числа {s1} расположены по возрастанию");
    }
    else
    {
        println!("Цифры числа {s1} не расположены по возрастанию");    
    }
}
