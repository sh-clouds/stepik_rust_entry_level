Циферки

Напишите программу, которая считывает целое число, а затем выводит его цифры через пробел и значность с новой строки, как показано в примерах.

Тестовые данные ✅

Sample Input 1:
123

Sample Output 1:
Число 123 состоит из цифр: 1 2 3
Число 123 является 3 значным

Sample Input 2:
12

Sample Output 2:
Число 12 состоит из цифр: 1 2
Число 12 является 2 значным

use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {
    let _s: String = input();
    let len = _s.len(); 
    let ind = _s.len() - 1;
    let s1 = &_s[..ind];    
    let s2 = &_s[ind..]; 
    print!("Число {_s} состоит из цифр: ");
    
    for ch in s1.chars() {
        print!("{ch} ");
    }
    println!("{s2}");     
    println!("Число {_s} является {len} значным");
}
