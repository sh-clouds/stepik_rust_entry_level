Стоп

Напишите программу, которая считывает стоп строку и вещественные числа, а после ввода сигнальной (стоп) строки выводит (до 1 знака) их сумму.

Гарантируется, что стоп строка не равна пробельным символам!

Тестовые данные ✅

Sample Input:
stop
1
2
3
stop

Sample Output:
6.0

Напишите программу. Тестируется через stdin → stdout
Верно решили 68 учащихся
Из всех попыток 37% верных



use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {
        let _s: String = input();
        let mut now: String;
        now = input();
        let mut sum = 0.0;

        while _s != now {
            let number: f64 = match now.trim().parse() {
                Ok(num) => num,
                Err(_) => continue,
        };
            
        sum += number;
        now = input();           
    }

    println!("{sum:.1}");     
}
