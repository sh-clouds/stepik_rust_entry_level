Количество установленных битов

Напишите программу, которая считывает целое число (u8) и выводит количество установленных битов в его двоичной записи в виде сообщения:Количество установленных битов в {:08b} равно {}.

Тестовые данные ✅

Sample Input:
1

Sample Output:
Количество установленных битов в 00000001 равно 1
Напишите программу. Тестируется через stdin → stdout
Верно решили 56 учащихся
Из всех попыток 83% верных


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {
    let n: u8 = input();
    let n_binary = &format!("{n:08b} ");
    let s1 = n_binary.to_string();
    let mut cnt: i32 = 0;
    for ch in s1.chars() {
        if ch == '1'
        {
            cnt +=1;
        }
    }

    println!("Количество установленных битов в {n:08b} равно {cnt}");     
}
