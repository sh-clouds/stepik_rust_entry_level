Числовой палиндром

Напишите программу, которая считывает натуральное число (u32) и выводит, является ли оно палиндром.

Если является, вывести в виде сообщения: Число {} является палиндромом;
Иначе: Число {} не является палиндромом.
Числовой палиндром - натуральное число, которое читается слева направо и справа налево одинаково.

Тестовые данные ✅

Sample Input:
121

Sample Output:
Число 121 является палиндромом

Напишите программу. Тестируется через stdin → stdout
Верно решили 57 учащихся
Из всех попыток 78% верных


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {
    let n: u32 = input();
    let s1 = n.to_string();
    let s2: String = s1.chars().rev().collect();
    
    if s1 == s2
    {
        println!("Число {s1} является палиндромом")
    }
    else
    {
        println!("Число {s1} не является палиндромом")
    }
}
