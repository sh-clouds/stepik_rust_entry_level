Быстрый логарифм

Напишите программу, которая считывает натуральное число b (u16) и выводит значение 
log2(b) используя цикл и битовые операции.

Тестовые данные ✅

Sample Input:
2

Sample Output:
1

Напишите программу. Тестируется через stdin → stdout

use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {
    let mut b: u16 = input();
    let mut a: u16 = 1;  
    let mut cnt: u16 = 0;
  
    while a < b
    {
        a = a << 1;
        cnt += 1;
    }  
    
    println!("{cnt}");
}
