Недостающее число

Напишите программу, которая считывает натуральные числа (u32), пока не будет введено иное, а затем выводит недостающее число введенной последовательности в виде сообщения: Пропущено число {}.

Гарантируется, что количество чисел в последовательности всегда на 1 меньше и стоп строка не равна пробельным символам!

Тестовые данные ✅

Sample Input:
1
2
3
5
1.5

Sample Output:
Пропущено число 4

Напишите программу. Тестируется через stdin → stdout
Верно решили 43 учащихся
Из всех попыток 35% верных



use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {
    let mut now: String;
    now = input();
    let mut sum = 0;
    let mut sum2: u32 = 0;   
    let mut f: bool = true;    
    let mut cnt: u32 = 0;
  
    let mut start: u32 = 0;    
    let mut end: u32 = 0;    
    
    while f == true 
    {
        let number: u32 = match now.trim().parse() 
        {
            Ok(num) => num,
            Err(_) => break,
        };

       if start >= number
       {
           start = number; 
       }
        
       if end <= number
       {
           end = number; 
       }  
       sum += number;
       now = input(); 
       cnt += 1
    }
    
    for i in (start..= end) {
        sum2 += i;           
    }   
            
    //println!("{sum:.1}");   
    //println!("{sum2:.1}"); 
    
    if sum != sum2
    {
        println!("Пропущено число {}", sum2 - sum);
    }
    else
    {
        println!("Пропущено число {}", end + 1);
    }
}
