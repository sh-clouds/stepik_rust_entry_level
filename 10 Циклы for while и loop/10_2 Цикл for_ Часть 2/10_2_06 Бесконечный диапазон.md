Бесконечный диапазон

Для создания диапазона чисел, начиная с указанного значения и до бесконечности, используется выражение start..:

fn main() {
    for i in -10.. {
        println!("{i}");
    }
}

Важно, чтобы цикл в какой-то момент завершился, чтобы не получился бесконечный цикл (зацикливание). Как это можно сделать, вы узнайте в уроке 10.5.



Шаг диапазона

Хотя тему методов мы ещё не прошли, но раз затронули эту тему в начале для ввода данных, почему бы не рассмотреть и эту тему 😁. Поскольку диапазон представлен структурой, значит у него должны быть некоторые полезные методы. Одним из таких является метод step_by, регулирующий шаг диапазона. Чтобы им воспользоваться, достаточно обернуть наш диапазон круглыми скобками и с помощью оператора доступа к элементу . вызвать step_by и в скобках указать нужный шаг:

fn main() {
    for i in (0..10).step_by(2) {
        println!("{i}");
    }
}

Игровая площадка ⛱️

Скобки вокруг 0..10 обеспечивают обращение не к 10, а к структуре Range. Особенностью метода step_by является то, что шаг должен быть строго больше нуля.



Метод rev

Ещё одним полезным методом является rev(), меняющий направление итератора. Обычно итераторы выполняют проход слева направо, но если мы хотим начать с конца, достаточно использовать rev():

fn main() {
    for i in (0..10).rev() {
        println!("{i}");
    }
}

Игровая площадка ⛱️

Использование rev() возможно только с конечными итераторами, реализующими трейт DoubleEndedIterator, так как только такие итераторы могут иметь определённый конец, с которого можно начать итерацию в обратном порядке.
