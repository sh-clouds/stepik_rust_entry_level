Циклический сумматор

Напишите программу, которая считывает натуральное число n (u8) и столько же вводимых пользователем чисел, а затем выводит (до 1 знака) их сумму, как показано в примерах.

Тестовые данные ✅

Sample Input 1:
1
1.5

Sample Output 1:

Сумма считанных чисел равна: 1.5


Sample Input 2:
2
-1.5
-2.7

Sample Output 2:

Сумма считанных чисел равна: -4.2

use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {  
    let mut n: u8 = input();  
    let mut f: f64;
    let mut sum: f64 = 0.0;
    
    for i in 0..n {
        f = input();
        sum += f;
    }        
    println!("Сумма считанных чисел равна: {:.1}", sum); 
}
