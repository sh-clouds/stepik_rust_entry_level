Произведение

Напишите программу, которая считывает три целых числа start (i32), end (i32) и step (usize) и выводит (до 1 знака) для диапазона start..=end с шагом step произведение его чисел.

Тестовые данные ✅

Sample Input:
-30
-10
3

Sample Output:
-1322697600.0

Напишите программу. Тестируется через stdin → stdout

Из всех попыток 40% верных


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {  
    let mut start: i32 = input();     
    let mut end: i32 = input(); 
    let mut step: usize = input();     
    let mut multy: f64 = 1.0;
    
    for i in (start..= end).step_by(step) {
        multy *= i as f64;
    }        
    println!("{multy:.1}"); 
}
