Числовой треугольник

Напишите программу, которая считывает натуральное число n (u8) и выводит числовой треугольник, состоящий из последовательности строк. В каждой строке числа увеличиваются по порядку, начиная с 1, и добавляется одно число с каждой новой строкой.

Тестовые данные ✅

Sample Input:
3

Sample Output:
1
1 2
1 2 3

Напишите программу. Тестируется через stdin → stdout

use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() 
{     
    let mut n: u8 = input();
    //n = 5;    
    
    for i in 1..=n   
    {    
        for j in 1..=i
        {
            print!("{j}");
            if j<i
            {
                print!(" ");
            }                        
        }
        println!("");   
    }    
}
