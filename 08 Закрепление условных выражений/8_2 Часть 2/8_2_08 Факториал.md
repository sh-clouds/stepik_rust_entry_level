Факториал

Дан факториал n! = 1 * 2 * 3 ... * n. Напишите программу, которая считывает натуральное число n (u32) и выводит последнюю цифру факториала. Так, при:

1! = 1 и последняя цифра равна 1;
2! = 1 * 2 = 2 и последняя цифра равна 2;
3! = 1 * 2 * 3 = 6 и последняя цифра равна 6;
4! = 1 * 2 * 3 * 4 = 24 и последняя цифра равна 4;
И т.д.

Вывод сделать в виде сообщения. К примеру, если ввели 4, то напечатать: Для 4! последняя цифра равна 4.

Тестовые данные ✅

Sample Input:
1

Sample Output:
Для 1! последняя цифра равна 1

Напишите программу. Тестируется через stdin → stdout
Верно решил 71 учащийся
Из всех попыток 59% верных


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {    
    let n: i32 = input(); 
    let last_num = if n == 1 {
        1
    } else if n == 2 {
        2
    } else if n == 3 {
        6
    } else if n == 4 {
        4
    } else {
        0
    };
    println!("Для {n}! последняя цифра равна {last_num}");
}
