Напишите программу, которая считывает количество часов (u8) отработанных за неделю, а затем выводит (до 2 знаков) заработную плату до вычетов и после, а также сумму налогов, учитывая:

базовый почасовой тариф: 1500 руб/час.
переработку более 40 часов в неделю: базовый почасовой тариф * 1.5.
налоговая ставка: 13%.
Так, если было введено 40 часов, то вывод будет следующим:

Заработная плата до вычетов: 60000.00 руб
Сумма налогов: 7800.00 руб
Заработная плата после вычетов: 52200.00 руб

Тестовые данные ✅
Sample Input:
40

Sample Output:
Заработная плата до вычетов: 60000.00 руб
Сумма налогов: 7800.00 руб
Заработная плата после вычетов: 52200.00 руб

Напишите программу. Тестируется через stdin → stdout
Верно решили 78 учащихся
Из всех попыток 47% верных


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {    
    let mut i1: u8 = input();
    let mut add = 0.0;

    if i1>40 {add = i1 as f64 - 40.0}
    let mut zp = i1 as f64 * 1500.0 + add as f64 * 750.0;
    let mut nalog = zp * 0.13;
    let mut itogo = zp - nalog;
    
    println!("Заработная плата до вычетов: {zp:.2} руб");
    println!("Сумма налогов: {nalog:.2} руб");
    println!("Заработная плата после вычетов: {itogo:.2} руб");      
}
