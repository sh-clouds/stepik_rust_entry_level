Коллинеарность точек

Напишите программу, которая считывает шесть целых чисел, представляющих собой координаты точек: P(x1, y1), A(x2, y2) и B(x3, y3), а затем выводит коллинеарны ли заданные точки.

Если коллинеарны вывести: Точки коллинеарны.
Иначе вывести: Точки не коллинеарны.
Коллинеарные точки - это три и более точек, которые лежат на одной прямой.

Тестовые данные ✅

Sample Input:
0
0
1
1
2
2

Sample Output:
Точки коллинеарны

Напишите программу. Тестируется через stdin → stdout

Из всех попыток 53% верных


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {    
    let x1: i32 = input();
    let y1: i32 = input();   
    let x2: i32 = input();   
    let y2: i32 = input();
    let x3: i32 = input();   
    let y3: i32 = input();      

    if ((x2-x1)*(y3-y1)-(x3-x1)*(y2-y1)) == 0
    {
        println!("Точки коллинеарны");        
    }
    else
    {
        println!("Точки не коллинеарны");       
    }     
}
