Парадоксальное выражение

Дан знакочередующийся ряд натуральных чисел: S = 1 – 2 + 3 – 4 + 5 – 6 + 7 – 8 + ... ± n.

Напишите программу, которая считывает натуральное число n и выводит сумму данного ряда. Так, при:

n = 1, S = 1;
n = 2, S = 1 - 2 = -1;
n = 3, S = 1 - 2 + 3 = 2;
И т.д.

Тестовые данные ✅

Sample Input:
1

Sample Output:
1

Напишите программу. Тестируется через stdin → stdout
Верно решили 72 учащихся
Из всех попыток 84% верных


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {    
    let n: i32 = input();
    let mut sum = 0;
      
    for i in 1..n+1
	{
		if i % 2 == 0 { sum = sum - i; }
        else { sum = sum + i; }
	}
    println!("{sum}");
}
