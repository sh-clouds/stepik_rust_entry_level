Last Digit

Напишите программу, которая считывает показатель степени (u8) числа 9 и выводит его последнюю цифру без возведения, как показано в примерах.

Так, если была считана 1, программа должна вывести: Последняя цифра 9 в степени 1 равна 9, поскольку 
9**1 равно 9. Если была считана 2, то программа должна вывести: Последняя цифра 9 в степени 2 равна 1, поскольку 
9**2 равно 81. И т.д.

Тестовые данные ✅

Sample Input:
1

Sample Output:
Последняя цифра 9 в степени 1 равна 9

Напишите программу. Тестируется через stdin → stdout
Верно решили 86 учащихся
Из всех попыток 73% верных


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {  
    let i: i32 = input(); 
    if i %2 ==0 { println!("Последняя цифра 9 в степени {i} равна 1"); }
    else { println!("Последняя цифра 9 в степени {i} равна 9"); }
}
