Разведчик

Два друга играли в разведчиков и кодировали сообщения собственным шифром. Фрагмент кодовой таблицы приведен ниже:

О   Н   Г   И
1   2   3   4

Напишите программу, которая считывает четырехзначное целое число и выводит расшифрованное сообщение. Гарантируется, что буквы в нем не повторяются.

Тестовые данные ✅

Sample Input:
2134

Sample Output:
НОГИ

use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {
    let arr = ["","О","Н","Г","И"];    
    let mut i: i32 = input(); 
    let i1 = i / 1000;
    i = i - i1 * 1000;
    let i2 = i / 100;
    i = i - i2 * 100;
    let i3 = i / 10;
    let i4 = i % 10;       
    println!("{}{}{}{}",arr[i1 as usize],arr[i2 as usize],arr[i3 as usize],arr[i4 as usize]); 
}
