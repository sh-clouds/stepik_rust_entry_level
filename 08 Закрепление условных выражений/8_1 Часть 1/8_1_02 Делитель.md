Делитель

Напишите программу, которая считывает четыре целых числа (x, a, b, и c) и выводит, является ли x делителем чисел a, b и c одновременно, как показано в примерах.

Если является вывести: {x} является делителем чисел {a}, {b}, {c}.
Иначе вывести: {x} не является делителем всех чисел.

Тестовые данные ✅

Sample Input:
2
4
6
8

Sample Output:
2 является делителем чисел 4, 6, 8

Напишите программу. Тестируется через stdin → stdout
Верно решили 88 учащихся
Из всех попыток 78% верных


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {
    let i1: i32 = input(); 
    let i2: i32 = input(); 
    let i3: i32 = input(); 
    let i4: i32 = input();     
    if i2 % i1 == 0 && i3 % i1 == 0 && i4 % i1 == 0 
    {
        println!("{i1} является делителем чисел {i2}, {i3}, {i4}");  
    }
    else
    {
        println!("{i1} не является делителем всех чисел");       
    }
}
