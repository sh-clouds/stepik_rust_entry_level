Дата

Напишите программу, которая считывает три положительных целых числа (день 1-31 u8, месяц 1-12 u8 и год 1-2024 u16) и выводит могут ли считанные числа представлять календарную дату, как показано в примерах.

Если могут вывести: Дата корректна!
Иначе вывести: Дата некорректна!
Учитывайте, что в зависимости от года (високосный или нет) количество дней в некоторых месяцах также меняется.

Тестовые данные ✅
Sample Input:
1
1
1

Sample Output:
Дата корректна!

Напишите программу. Тестируется через stdin → stdout
Верно решили 74 учащихся
Из всех попыток 58% верных


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {    
    let i1: u8 = input();
    let i2: u8 = input();
    let i3: u16 = input();   
    let mut m = [31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31];
  
    if (i3 % 4 == 0) && (i3 % 100 != 0) || (i3 % 400 == 0)
    {
        m = [31, 29, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31];
    }
    
    if i1 <= m[(i2-1) as usize] { println!("Дата корректна!"); } 
    else { println!("Дата некорректна!"); }     
}
