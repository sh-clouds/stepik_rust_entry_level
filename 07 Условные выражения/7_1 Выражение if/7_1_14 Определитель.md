Определитель 

Напишите программу, которая считывает три целых числа и определяет, различны ли они. В случае истины выводит Числа различны, в противном случае выводит Числа неразличны.

Будем считать числа неразличными, если есть хотя бы два одинаковых.

Тестовые данные ✅

Sample Input 1:
1
2
3

Sample Output 1:
Числа различны
Sample Input 2:

10
100
10

Sample Output 2:
Числа неразличны
Напишите программу. Тестируется через stdin → stdout
Верно решили 148 учащихся
Из всех попыток 75% верных


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {   
    let f1: f64 = input();
    let f2: f64 = input();
    let f3: f64 = input();    
    
    if f1 == f2 || f2 == f3 || f1 == f3
    {
        println!("Числа неразличны");              
    }
    else 
    {
        println!("Числа различны");               
    }    
}
