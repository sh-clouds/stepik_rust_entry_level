Високосный год

Напишите программу, которая считывает целое число (год) и определяет является он високосным. Если год является високосным, то выведите сообщение {year} является високосным годом, в противном случае выведите {year} не является високосным годом, как показано в примере.

Год является високосным, если он кратен 4, за исключением вековых лет (оканчивающихся на 00). Вековой год является високосным только в том случае, если он кратен 400.

Тестовые данные ✅

Sample Input:
2024

Sample Output:
2024 является високосным годом

Напишите программу. Тестируется через stdin → stdout
Верно решили 146 учащихся
Из всех попыток 48% верных


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {   
    let i1: i32 = input();  
    
    if (i1 % 4 == 0) && (i1 % 100 != 0) || (i1 % 400 == 0)
    {
        println!("{} является високосным годом", i1);              
    }
    else 
    {
        println!("{} не является високосным годом", i1);                
    }    
}
