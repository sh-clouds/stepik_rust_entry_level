Минус и Плюс

Напишите программу, которая считывает вещественное число, определяет, является оно положительным или отрицательным, и выводит (до 1 знака) результат, как показано в примере. Поскольку 0 не является ни отрицательным, ни положительным, то в тестах его нет.

Тестовые данные ✅

Sample Input:
10

Sample Output:
Число 10.0 является положительным

Напишите программу. Тестируется через stdin → stdout
Верно решили 187 учащихся
Из всех попыток 61% верных


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {
    let f1: f64 = input(); 
    if f1 > 0.0 {
        println!("Число {:.1} является положительным", f1);  
    }
    else {
        println!("Число {:.1} является отрицательным", f1);          
    }
}
