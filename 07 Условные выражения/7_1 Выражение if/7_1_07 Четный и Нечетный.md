Четный и Нечетный

Напишите программу, которая считывает два целых числа и определяет, какое является четным, а какое нечетным.

Тестовые данные ✅

Sample Input:
1
2

Sample Output:
Число 1 является нечетным
Число 2 является четным

Напишите программу. Тестируется через stdin → stdout

use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {
    let i1: i32 = input(); 
    let i2: i32 = input(); 
    let mut s1 = "";
    let mut s2 = "";
    
    if i1 % 2 == 0 {s1 = "четным";} else {s1 = "нечетным";}
    if i2 % 2 == 0 {s2 = "четным";} else {s2 = "нечетным";}    
    println!("Число {} является {}", i1, s1);
    println!("Число {} является {}", i2, s2);    
}
