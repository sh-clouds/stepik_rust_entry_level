Терморегулятор 

Вы инженер космических ракет. Вам необходимо написать программу для терморегулятора, которая будет принимать текущую температуру (°C, f64) в отсеке и выводить (до 1 знака) соответствующее сообщение, как показано в примере:

Если текущая температура ниже установленной нормы 24.0°C, программа должна выводить: Температура {}°C ниже нормы 24.0°C, включаю отопление.
Если текущая температура выше установленной нормы 28.0°C, программа должна выводить: Температура {}°C выше нормы 28.0°C, отключаю отопление.
Если текущая температура в пределах нормы, программа должна выводить: Температура {}°C в пределах нормы.
Только ваша программа сможет помочь поддерживать оптимальный температурный режим на станции, обеспечивая комфорт для экипажа и сохранность оборудования.

Тестовые данные ✅

Sample Input:
21.5

Sample Output:
Температура 21.5°C ниже нормы 24.0°C, включаю отопление

Напишите программу. Тестируется через stdin → stdout

use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {   
    let mut f1: f64 = input();
    //f1 = 29.0;
    
    if f1 > 28.0
    {
        println!("Температура {:.1}°C выше нормы 28.0°C, отключаю отопление", f1);        
    }
    else if f1 < 24.0
    {
        println!("Температура {:.1}°C ниже нормы 24.0°C, включаю отопление", f1);
    } 
    else
    {
        println!("Температура {:.1}°C в пределах нормы", f1);          
    }     
}
