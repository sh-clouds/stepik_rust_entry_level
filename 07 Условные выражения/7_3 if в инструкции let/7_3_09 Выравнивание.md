В редакторе кода представлен массив значений. Дополните код и выведите отсортированный по возрастанию массив значений с помощью спецификатора :?.

Сортировку массива необходимо сделать c помощью условных выражений.

Sample Input:

Sample Output:
[-2, -1, 3, 5, 8]

Напишите программу. Тестируется через stdin → stdout
Верно решили 100 учащихся
Из всех попыток 85% верных

use std::io;

fn main() {
    let mut array = [5, -2, 8, -1, 3];
    for i in 1..array.len() {
        for i in 1..array.len() {
            if array[i] < array[i-1] {
                (array[i-1], array[i]) = (array[i], array[i-1]);
            }
        }
    }    
    println!("{:?}", array); 
}
