Тонометр

Напишите программу, которая считывает возраст (u8) и два целых (u8) числа являющиеся показателями медицинского прибора для измерения артериального давления (АД):

SYS - показывает систолическое (верхнее) давление.
DIA - показывает диастолическое (нижнее) давление.
Программа должна вывести следующее:

Если показатель(и) в норме, то напечатать:
Систолическое АД в норме
или
Диастолическое АД в норме
или
Систолическое и Диастолическое АД в норме.
В противном случае вывести на сколько завышен или занижен каждый показатель:
Систолическое АД {} ниже нормы на {} или Систолическое АД {} выше нормы на {}
Диастолическое АД {} ниже нормы на {} или Диастолическое АД {} выше нормы на {}.
Для наглядности посмотрите тестовые данные. Гарантируется, что возраст в заданном диапазоне.

Возрастные группы	  Систолическое  АД	  Диастолическое АД
18-39 лет	          90-139	          60-89
40-59 лет	          91-149	          61-91
60+ лет	              91-159	          61-91

Тестовые данные ✅

Sample Input:
18
90
60

Sample Output:
Систолическое и Диастолическое АД в норме

Напишите программу. Тестируется через stdin → stdout
Верно решил 91 учащийся
Из всех попыток 36% верных


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {    
    let u1: u8 = input();    
    let u2: u8 = input();
    let u3: u8 = input(); 
    
    let mut s1 = 91;  //60+       
    let mut s2 = 159;
    let mut d1 = 61;        
    let mut d2 = 91;
    
    if u1 >= 18 && u1 <= 39 
    {
        s1 = 90;        
        s2 = 139;
        d1 = 60;        
        d2 = 89;        
    }
    else if u1 >= 40 && u1 <= 59 
    {
        s1 = 91;        
        s2 = 149;
        d1 = 61;        
        d2 = 91;                  
    }
    
    if (u2 >= s1 && u2 <= s2) && (u3 >= d1 && u3 <= d2) 
        { println!("Систолическое и Диастолическое АД в норме") }
    
    else
    {
        if (u2 >= s1 && u2 <= s2) 
            { println!("Систолическое АД в норме") }
        else if u2 < s1
            { println!("Систолическое АД {} ниже нормы на {}", u2, s1 - u2) }
        else 
            { println!("Систолическое АД {} выше нормы на {}", u2, u2 - s2) }         

        if (u3 >= d1 && u3 <= d2) 
            { println!("Диастолическое АД в норме") }
        else if u3 < d1
            { println!("Диастолическое АД {} ниже нормы на {}", u3, d1 - u3) }
        else 
            { println!("Диастолическое АД {} выше нормы на {}", u3, u3 - d2) }     
    } 
}
