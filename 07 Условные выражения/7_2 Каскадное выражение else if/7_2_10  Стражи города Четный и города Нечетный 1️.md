Стражи города Четный и города Нечетный

Существуют два древних города: город Четный и город Нечетный. Жители каждого города славятся своей уникальностью. В городе Четный все построено на порядке и симметрии, а его стражники строго следят за тем, чтобы только четные числа входили в город. Стражники города Нечетный, напротив, предпочитают хаос и случайность, и они впускают нечетные числа.

Недавно в обоих городах возникла проблема: стражам стало не хватать времени на проверку входящих в город чисел. Для облегчения работы они обратились к вам с просьбой помочь им с определением, какое число пропускать в город, а какое нет.

Напишите программу, которая считывает название города и два целых числа  и определяет, разрешен ли им проход в город или нет, как показано в примере.

Тестовые данные ✅

Sample Input:
Четный
1
2

Sample Output:
1 в город Четный вход запрещен
2 в город Четный вход разрешен

use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {    
    let s1: String = input();
    let i1: i32 = input();    
    let i2: i32 = input();
    
    if s1 == "Четный"
    {
        if i1 % 2 == 0 { println!("{} в город Четный вход разрешен", i1) }
        else { println!("{} в город Четный вход запрещен", i1) }   
                
        if i2 % 2 == 0 { println!("{} в город Четный вход разрешен", i2) }
        else { println!("{} в город Четный вход запрещен", i2) }      
    }
    else
    {
        if i1 % 2 == 0 { println!("{} в город Нечетный вход запрещен", i1) }
        else { println!("{} в город Нечетный вход разрешен", i1) }   
                
        if i2 % 2 == 0 { println!("{} в город Нечетный вход запрещен", i2) }
        else { println!("{} в город Нечетный вход разрешен", i2) }  
    }    
}
