Кассовый Компаньон 1.0: Сканер Монет

Вы изобретатель, создающий новый механический торговый автомат, который принимает монеты различного достоинства. Однако перед тем как ваш автомат попадет на рынок, вам необходимо написать программу для монетоприемника, чтобы убедиться, что он правильно распознает монеты.

Напишите программу для монетоприемника механического торгового автомата, которая считывает целое число и определяет, является ли оно монетой, принимаемой вашим автоматом. Если введенное значение соответствует одной из монет (1, 2, 5 или 10), программа должна вывести сообщение: Принята монета номинала {}. В противном случае программа должна вывести Монеты такого номинала не принимаются.

Тестовые данные ✅

Sample Input:
1

Sample Output:
Принята монета номинала 1

Напишите программу. Тестируется через stdin → stdout
Верно решили 133 учащихся
Из всех попыток 85% верных


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {   
    let i1: i32 = input();    
    
    if (i1 == 1) || (i1 == 2) || (i1 == 5) || (i1 == 10)
    {
        println!("Принята монета номинала {}", i1);        
    }
    else
    {
        println!("Монеты такого номинала не принимаются")          
    }  
}
