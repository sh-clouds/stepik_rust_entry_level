Среды разработки

У Rust есть официальный сайт  и средства разработки: пакетный менеджер и компилятор. Для начинающих программистов рекомендуем использовать интегрированную среду разработки, облегчающую редактирование, сборку, тестирование и упаковку программного обеспечения.

Вашему вниманию предлагаются следующие программные решения, которые являются наиболее удобными и пользуются достаточно большой популярностью как среди профессионалов, так и среди новичков.

Интегрированные среды разработки

- Free

Visual Studio Code

Visual Studio Code поставляется компанией Microsoft и является универсальным решением почти для всех языков программирования. Ещё одним плюсом данной программы является возможность использовать её в коммерческой сфере, т.е. в компании, командной разработке и т.д. VSCodium  - свободная альтернатива Visual Studio Code.

      

NeoVim/Vim


Другим хорошим решением является использование редакторов текста с поддержкой программного кода, например, NeoVim.

NeoVim — это текстовый редактор на базе Vim, разработанный с учётом расширяемости и удобства использования. Установка и настройка может вызвать трудности у новичков.
      

Replit

Если вы не хотите устанавливать программу, то можете использовать онлайн интегрированную среду разработки. Одним из таких решений является сервис Replit. Для того чтобы им воспользоваться,  достаточно зайти на их сайт, зарегистрироваться и пользоваться.

- Free for private 

RustRover

IDE RustRover от компании JetBrains. Продукты данной компании пользуются большой популярностью. Частные лица, желающие использовать RustRover в некоммерческих целях, могут использовать IDE бесплатно. В противном случае необходимо будет приобрести лицензию, как и для других продуктов JetBrains.

      

Visual Studio

Visual Studio также поставляется компанией Microsoft. Если вы не хотите заморачиваться с настройкой программной среды, то можете ставить данную программу. Программа поставляется в трёх версиях. Версия Community является бесплатной для индивидуальных разработчиков. Это означает, что вы также можете создавать ею платные продукты.



Xcode


Xcode подойдёт для тех, кто пользуется продуктами от компании Apple. Если вы работайте на Macbook и хотите использовать встроенные решения, то можете скачать данную программу из официального магазина App Store. Но может потребоваться дополнительная настройка, поэтому лучше используйте Zed.

Любители программировать в терминале могут напрямую использовать либо Cargo систему сборки и менеджера пакетов Rust, либо компилятор rustc. В этом случае рекомендуем использовать Cargo.


Поставляемые средства разработки
 - Free 

Cargo

Мощная система сборки и менеджер пакетов Rust. Cargo обеспечивает загрузку зависимостей пакета Rust, их компиляцию, создание распространяемых пакетов и загрузку в реестр пакетов сообщества Rust crates.io.

      

rustc

Официальный компилятор языка. Использует LLVM для оптимизации и перевода высокоуровневого кода Rust в низкоуровневый машинный код.

      

gccrs (early stage)


Альтернативна rustc на GCC. Полная реализация языка Rust поверх GCC. Компилятор находится на очень ранней стадии и пока не пригоден для компиляции реальных программ на Rust.
