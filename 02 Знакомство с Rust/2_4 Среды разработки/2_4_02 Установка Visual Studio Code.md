Установка Visual Studio Code

Мы решили, что вам будет удобнее решать практические задачки, используя IDE, и затем выкладывать на проверку в Степике. Поэтому покажем, как его установить и настроить. В качестве IDE был выбран Visual Studio Code. Сторонники открытых решений могут использовать VSCodium .

Windows 11 
Шаг 1: Скачивание Visual Studio Code
Откройте окно браузера и перейдите на сайт code.visualstudio.com;
Кликните по синей кнопке Windows.


Шаг 2: Запуск установщика
Запустите скачанный установщик;
Отметьте пункт I accept the agreement;
Следуйте указаниям установщика, нажимая кнопку Next;


Выберите папку для установки;
Если не хотите создавать папку в меню Пуск, можете поставить галочку;
Установите галочку напротив данных пунктов; 


 Шаг 3: Установка расширений для кода
Откройте Visual Studio Code


 Откройте расширения​​​​​​


 В строке поиска напишите rust-analyzer. Найдите и установите данное расширение

 Теперь установите расширение Code Runner


Зайдите в настройки


В строке поиска напишите Code-runner:Run in Terminal и поставьте галочку напротив Whether to run code in Integrated Terminal


Шаг 4: Установка компилятора Rust 
Скачайте и установите набор инструментов языка Rust. В зависимости от битности вашей ОС.


Откроется терминальное окно, где необходимо будет ввести 1 и дождаться завершения установки



​​​​​​​Шаг 5: Проверка корректной работы
Создайте новый проект, введя в терминале команду  cargo new \путь\к\проекту​​​​
После введите команду code .\путь\к\проекту. Так, вы откройте свой проект в VSCode​​​​​​
Можно приступать к запуску тестовой программы


 Шаг 6: Запуск кода
Правой кнопкой мыши запустите код, нажав на Run Code

Ура всё готово. В терминале результат работы программы.
Если хотите, чтобы сохранение кода происходило автоматически при запуске, то поставьте галочку в соответствующем пункте code-runner.saveFileBeforeRun
