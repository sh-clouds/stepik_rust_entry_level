Программное обеспечение на Rust

В настоящее время некоторые компании представили программное обеспечение на Rust. Это стало возможным благодаря уникальным преимуществам языка, таким как безопасность, скорость и параллелизм. Хотя Rust ещё молод и не успел зарекомендовать себя в передовых и популярных программных решениях, которые в основном написаны на C или C++, но уже существуют примеры успешного использования Rust в различных проектах. 

Системное программное обеспечение
Операционные системы
Redox OS  - Unix-подобная операционная система, написанная на Rust, целью которой является воплощение инноваций Rust в современном микроядре и полном наборе приложений.
 

Theseus OS  - современная ОС, написанная с нуля на Rust, устраняющая семантический разрыв между компилятором и аппаратным обеспечением за счёт максимального использования возможностей языковой безопасности и аффинных типов.



Fuchsia OS  - операционная система общего назначения с открытым исходным кодом, частично написанная на Rust.
 

Bottlerocket  - операционная система с открытым исходным кодом на базе Linux, специально созданная Amazon Web Services для запуска контейнеров.

 

 

Surface UEFI - Surface и Project Mu совместно работают над внедрением Rust в экосистему UEFI. Project Mu внёс необходимые изменения в среду сборки UEFI, чтобы обеспечить плавную интеграцию модулей Rust в кодовые базы UEFI. Surface использует эту поддержку для создания модулей Rust в прошивке платформы Surface.

lilos  - маленькая операционная система реального времени для микроконтроллеров, использующая асинхронный интерфейс Rust.

hubris  - микроконтроллерная операционная среда, разработанная для глубоко встроенных систем с требованиями к надежности.

Драйвера устройств
rusticl  - разработчики Mesa объявили о сертификации организацией Khronos драйвера rusticl, признаный полностью совместимым со спецификацией OpenCL 3.0.

Kazan  - разрабатываемый драйвер Vulkan, который поддерживает кроссплатформенный программный рендеринг и является драйвером для графического процессора на базе RISC-V.

Vulkano  - Безопасная и функциональная оболочка Rust вокруг API Vulkan.

 

 



Nova  - драйвер GSP на Rust для графических процессоров Nvidia, призванный стать преемником Nouveau.

Встраиваемые системы
Infineon Technologies расширила поддержку языка программирования Rust посредством HighTec EDV-Systeme.
 

Rust теперь доступен для операционной системы реального времени и гипервизора PikeOS.

Sensirion недавно использовали Rust для создания демонстрации Particulate Matter Sensor.

blethrs  - загрузчик Ethernet для STM32, написанный на Rust.
 

49Nord использовал Rust для доставки программного обеспечения быстрее и более надежно, чем они ожидали. Rust обеспечил им гарантированную безопасность памяти, что является критическим аспектом в разработке высоконадёжного программного обеспечения.
 

Робототехника
Airhart используют Rust для всего встроенного программного обеспечения.

Прочее
COSMIC  - новая cреда рабочего стола для Linux от System76, написанная на Rust.

 

Firecracker  - безопасные и быстрые микровиртуальные машины для бессерверных вычислений.

 
Прикладное программное обеспечение
Серверные и сетевые решения
Передовая сервисная сеть Linkerd использует Rust для написания своих продуктов.
 

Dropbox активно интегрирует Rust в свою инфраструктуру. Rust лежит в основе недавно переписанного механизма синхронизации настольного клиента, а также используется в библиотеках для Capture.

Discord переписывает различные части своего программного обеспечения. Rust применяется в игровом SDK для захвата и кодирования видео в Go Live, Elixir NIF, нескольких серверных сервисах и т.д.

NPM внедрил Rust для устранения узких мест в производительности. Из-за нехватки производительности команда инженеров npm перешла с Node.js на Rust развернув новый сервис аутентификации.

GitHub выпустил переработанную поисковую систему по коду Blackbird на Rust. Blackbird может обрабатывать до 640 запросов в секунду по сравнению с 0.01 запроса в секунду для ripgrep и может индексировать со скоростью около 120 000 документов в секунду.

Mozilla перенесли инфраструктуру Push-соединений на Rust, обеспечив одновременную обработку до 20 миллионов сетевых подключений в часы пик с серверами на Rust.

AFFiNE  - альтернатива Notion и Miro, ориентированная на конфиденциальность, локальность, открытость и готовность к использованию. Использует Rust на стороне сервера.

SeaQL  - независимая организация, разрабатывающая инструменты с открытым исходным кодом, которые позволяют разработчикам создавать приложения на Rust.
 

Wasmer  - лидирующая среда выполнения Wasm, поддерживающая WASIX, WASI и Emscripten.

 

Wasmtime  - быстрая и безопасная среда выполнения для WebAssembly.
 

WASIX  - модульный системный интерфейс для WebAssembly (Wasm), который позволяет разработчикам запускать программы Wasm на любой платформе, включая браузер, облако и т.д.
 

WasmEdge  - легкая, высокопроизводительная и расширяемая среда выполнения WebAssembly для облачных, периферийных (Edge) и децентрализованных приложений. Поддерживает бессерверные приложения, встроенные функции, микросервисы, смарт-контракты и устройства IoT.

WasmCloud  - универсальная платформа приложений, которая помогает создавать и запускать глобально распределённые приложения WebAssembly в любом облаке или на периферии.


Fermyon  - легкое, быстрое и по-настоящему бессерверное облако, созданное для компиляции и отправки вашего кода в виде двоичных файлов Wasm.

 

quickwit  - облачный поисковый движок для обеспечения наблюдаемости. Открытая альтернатива Datadog, Elasticsearch, Loki и Tempo.

ParadeDB  - современная альтернатива Elasticsearch, созданная на базе Postgres.


GlareDB  - быстрая база данных SQL для анализа распределенных данных. Подключайте источники данных и выполняйте запросы к ним с исключительной производительностью.
 

Fluvio  - это легкая высокопроизводительная распределённая система потоковой передачи данных, написанная на Rust и WebAssembly.
 

Meilisearch  - молниеносная поисковая система, которая легко встраивается в приложения, веб-сайты и рабочий процесс.
 

Qdrant (quadrant)  - высокопроизводительная крупномасштабная векторная база данных для ИИ нового поколения. Также доступна в облаке.
 

Neon  - бессерверная альтернатива AWS Aurora Postgres с открытым исходным кодом. Разделяет хранилище и вычисления, и заменяет уровень хранения PostgreSQL, перераспределяя данные по кластеру узлов.
 

SurrealDB  - комплексная облачная база данных, предназначенная для современных приложений, включая веб-приложения, мобильные, бессерверные приложения, Jamstack, серверные и традиционные приложения.
 

Databend  - облачное хранилище данных с открытым исходным кодом написанное на Rust, которое служит экономичной альтернативой Snowflake. Быстрое выполнение запросов и приём данных, комплексный анализ крупнейших в мире наборов данных.

InfluxDB  - база данных с открытым исходным кодом оптимизированный для быстрого и высокодоступного хранения и извлечения данных временных рядов.
 

Tikv  - распределённая транзакционная база данных типа "ключ-значение", изначально созданная в дополнение к TiDB.
 

Greptime  - единая облачная база данных временных рядов с открытым исходным кодом для метрик, журналов и событий с поддержкой PromQL/SQL/Python. Доступно на GreptimeCloud.
 

Arroyo  -  движок распределённой потоковой обработки, написанный на Rust и предназначенный для эффективного выполнения вычислений с учётом состояния потоков данных.
 

Prisma engines  - движки, которые обеспечивают работу основного стека Prisma, в первую очередь Prisma Client и Prisma Migrate написаны на Rust.
 

Rivet  - запуск и масштабирование приложений реального времени. Rivet Actors имеют встроенные RPC, состояние и события - самый простой способ создания современных приложений.
 

Youki  - контейнерная среда выполнения в Rust. Аналог runc.

 

Веб-разработка
Mako  - новый веб-упаковщик промышленного уровня на Rust, для веб-приложений, библиотек и фреймворков. Разработан, чтобы быть быстрым, надёжным и простым в использовании.

Rsbuild  - высокопроизводительный инструмент сборки. Интегрирует высокопроизводительные инструменты на основе Rust от сообщества, включая Rspack , SWC и Lightning CSS для обеспечения первоклассной скорости сборки и удобства разработки.

Farm  - чрезвычайно быстрый Vite - совместимый инструмент для веб-сборки, написанный на Rust.

 

Vite  - обнародовал долгосрочный план Vite по работе над Rolldown, Rust-портом Rollup с совместимыми API.

 

Turbopack  - инкрементный сборщик и система сборки, оптимизированная для JavaScript и TypeScript, написанная на Rust. Преемник Webpack на базе Rust.

WinterJS  - сервер JavaScript Service Workers, написанный на Rust, который использует движок SpiderMonkey для выполнения JavaScript (тот же движок, который использует Firefox).

Deno  - современная среда выполнения JavaScript и TypeScript.

 

tailwindcss  - перенесли некоторые из самых дорогих и распараллеливаемых частей фреймворка на Rust, сохранив при этом ядро ​​фреймворка на TypeScript для обеспечения расширяемости.

Oxlint  - коллекция высокопроизводительных инструментов для JavaScript и TypeScript, написанных на Rust.
 

Biome  - набор инструментов для веб-проектов, призванный предоставить функциональные возможности для их поддержки. Biome предлагает форматтер и линтер, которые можно использовать через CLI и LSP.

Parcel  - современный инструмент для сборки веб-проектов, который позиционируется как zero configuration build tool, что означает минимальную необходимость в настройке. Компилятор JavaScript Parcel написан на Rust.

SWC  - расширяемая платформа на базе Rust для нового поколения быстрых инструментов разработки.

Lightning CSS  - чрезвычайно быстрый парсер, преобразователь, сборщик и минификатор CSS, написанный на Rust.
 

Robyn  - высокопроизводительный, управляемый сообществом и дружелюбный к инноваторам веб-фреймворк со средой выполнения Rust.
 

TurboVorderseite  - система веб-сборки для интернет-магазинов с использованием WebAssembly, полностью написанная на Rust с использованием фреймворка Leptos и TailwindCSS. Пример использования.

 

Настольное и мобильное ПО
Boa  - встраиваемый экспериментальный движок Javascript, написанный на Rust. В настоящее время  поддерживает некоторые языки.

Zed  - мощный многофункциональный редактор кода от создателей Atom и Tree-sitter.
 

Servo  - встраиваемый, независимый, безопасный для памяти, модульный, параллельный механизм веб-рендеринга, написанный на Rust, с поддержкой WebGL и WebGPU, адаптируемый к настольным, мобильным и встроенным приложениям.

Gosub  - браузерный движок с открытым исходным кодом, написанный на Rust.

 

Helix  - текстовый редактор, вдохновлённый Kakoune / Neovim и написанный на Rust.

 

 

xi-editor  - экспериментальный текстовый редактор с открытым исходным кодом, написанный на Rust. Разрабатывается с учётом производительности и современных методов проектирования.

Sniffnet  - инструмент сетевого мониторинга, помогающий легко и комфортно отслеживать интернет-трафик.

 

RustDesk  - программное обеспечение для удаленного доступа к рабочему столу, написанное на Rust. Альтернатива TeamViewer с открытым исходным кодом.

Yazi  - невероятно быстрый терминальный файловый менеджер, написанный на Rust и основанный на асинхронном вводе-выводе.

Warp  - cовременный терминал на базе Rust со встроенным искусственным интеллектом, позволяющий вам и вашей команде быстрее создавать отличное программное обеспечение

Alacritty  - быстрый, современный терминал, написанный на Rust. Призван предоставить высокую производительность и возможности настройки для пользователей.

Zellij  - терминальная среда со всем необходимым. Включает в себя множество встроенных функций, которые позволяют пользователям расширять его и создавать свою собственную персонализированную среду.
 

Tiny Glade - ламповая инди игра про строительство замков. Камешек к камешку, дощечка к дощечке – никаких битв, никакой возни с ресурсами или боязни ошибиться. Просто устройтесь поудобнее и стройте прекрасные замки на заросших лугах.
 

 

Инструменты командной строки
git-cliff  - генератор журналов изменений с широкими возможностями настройки, соответствующий спецификациям обычных коммитов.

ripgrep  - быстрый аналог GNU grep, поддерживающий Unicode, включённый по-умолчанию.

 

Ruff  - невероятно быстрый линтер и форматер кода Python, написанный на Rust.
uv  - невероятно быстрый пакетный менеджер Python, написанный на Rust.
 

Orogene  - менеджер пакетов нового поколения для инструментов, использующих node_modules, таких как сборщики пакетов, инструменты CLI и приложения на базе Node.js.

Brioche  - простой и гибкий менеджер пакетов и сборочный инструмент. Создан для максимального упрощения использования любых инструментов на любой платформе без виртуализации, совместно со всеми участниками и членами команды.

Ducker  - терминальное приложение для управления контейнерами Docker, вдохновленное K9s.

 

rust-command-line-utilities - список некоторых утилит командной строки.

Криптовалюта и блокчейн
Polkadot - блокчейн платформа и криптовалюта. Использует Rust для построения своей базовой инфраструктуры.
 

Elrond (EGLD) - один из революционных блокчейнов, ориентированный на скорость и эффективность за счет использования сегментирования в масштабе Интернета, также использует Rust.

Solana  - одна из популярных блокчейн сетей в среде Web3 с самыми быстрыми темпами роста, использующая язык программирования Rust.

Near  - ещё один популярный блокчейн, с особым упором на хранение данных на уровне сообщества, использующий язык программирования Rust.

Solidity  - новый компилятор для Solana и Polkadot, написанный на Rust.
