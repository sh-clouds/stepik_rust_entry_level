Различные фреймворки и библиотеки Rust

Каким бы хорошим не был язык, ему необходимы время и поддержка. Сообществом языка активно развиваются фреймворки и библиотеки, которые помогают разработчикам создавать надёжное и эффективное программное обеспечение на Rust, а некоторые ещё и на других языках. Эти инструменты предоставляют широкий спектр возможностей: от низкоуровневого программирования до веб и кроссплатформенной разработки.

Для разработки системного ПО
Операционные системы
Rust OSDev  - инструменты для разработки операционной системы на Rust.

 

operating-system - привязки к API-интерфейсам, специфичным для операционной системы.

 

Драйвера устройств
windows-drivers-rs  - платформа, позволяющая разрабатывать драйвера Windows на Rust.
 

Встраиваемые системы
Embassy  - фреймворк следующего поколения для встроенной разработки. Пишите безопасный, правильный и энергоэффективный встроенный код быстрее, используя язык программирования Rust, его асинхронные возможности и библиотеки Embassy.

Rust Embedded  - официальная рабочая группа Rust, занимающаяся созданием экосистемы инструментов для встроенной разработки, не зависящих от платформы.

RTIC  - фреймворк реального времени с прерываниями и параллельной обработкой для микроконтроллеров ARM Cortex-M.
 

Embedded-graphics  - библиотека 2D-графики, ориентированная на встроенные устройства с ограниченным объемом памяти.
 

sensirion-i2c-rs  - библиотека Rust с общим функционалом для датчиков на базе I2C от Sensirion.


embedded-hal-driver - crate's для встроенных устройств и устройств без операционной системы.

 

Робототехника
dora-rs  - платформа для ИИ и робототехнических приложений с низкой задержкой, компонуемостью и распределенным потоком данных.
 

Copper  - удобный фреймворк робототехники, предназначенный для создания быстрых и надежных роботов. Copper для роботов - то же, что игровой движок для игр.
 

RoboPLC  - фреймворк реального времени для создания микросервисов реального времени, ПЛК и промышленных роботов на Rust.
 

Rapier  - Быстрый 2D и 3D физический движок для языка программирования Rust.
 

Rerun  - SDK для визуализации мультимодальных данных с учётом их изменения во времени.

 

AeroRust  -  экосистема для аэрокосмических приложений.
 

Robotics - crate's связанные с робототехникой. Создавайте прошивки для роботов, дронов и автономных машин.


 

Прочее
AYA  - библиотека eBPF для Rust, созданная с упором на удобство и простоту использования разработчиками.

 

Для разработки прикладного ПО
Серверные и сетевые решения
Tokio  - асинхронная среда выполнения для Rust, предоставляющая необходимые инструменты для создания сетевых приложений, обеспечивая гибкость работы с различными системами: от крупных серверов с множеством ядер до небольших встроенных устройств.

smol  - небольшая и быстрая асинхронная среда выполнения для Rust.

 

futures-rs  - библиотека, обеспечивающая основы асинхронного программирования с нулевой стоимостью на Rust.

Loco (Rust on Rails)  - фреймворк Rust для проектов и стартапов, разрабатываемых в свободное время одним человеком.
 

Exograph  - серверный интерфейс для веб-приложений, мобильных приложений и приложений искусственного интеллекта.

Hyper  - быстрая и безопасная HTTP библиотека для Rust.

Tonic  - реализация gRPC поверх HTTP/2, ориентированная на высокую производительность, совместимость и гибкость. Эта библиотека была создана для первоклассной поддержки async/await и в качестве основного строительного блока для производственных систем на Rust.

reqwest  - простой и мощный Rust HTTP-клиент.

 

Iggy  - платформа постоянной потоковой передачи сообщений, написанная на Rust, поддерживающая транспортные протоколы QUIC, TCP, HTTP и способная обрабатывать миллионы сообщений в секунду.

 Zenoh  - сетевой протокол с нулевыми накладными расходами.

 

Iroh  - Набор инструментов для создания распределённых приложений.

Pavex  - простой в использовании фреймворк Rust для создания надёжных и производительных API.

 

Pingora  - платформа (библиотека), разработанная компанием Cloudflare на Rust для создания быстрых, надёжных и программируемых сетевых систем.

Rama  - модульная сервисная платформа для перемещения и преобразования сетевых пакетов.

 

DataFusion  - очень быстрый, расширяемый механизм запросов для создания высококачественных систем, ориентированных на данные, написан на Rust с использованием формата Apache Arrow в памяти.

Polars  - интерфейс DataFrame поверх механизма запросов OLAP, реализованный на Rust с использованием формата столбцов Apache Arrow в качестве модели памяти.

minitrace-rust  -  чрезвычайно быстрая библиотека трассировки для Rust.

Rustls  - современная библиотека TLS, написанная на Rust.
 

Serde  - фреймворк для эффективной и универсальной сериализации и десериализации структур данных Rust. Также разрабатывают и строго типизированную библиотеку JSON для Rust.

Diesel  -  безопасный, расширяемый ORM и конструктор запросов для Rust.

 

SQLx  - асинхронный крейт на чистом Rust и SQL, содержащий запросы проверяемые во время компиляции, без DSL. Поддерживает PostgreSQL, MySQL, MariaDB и SQLite.

Tonbo  - встраиваемая база данных типа "ключ-значение". Поддерживает хранение данных на локальных дисках и S3 для выполнения запросов в любом месте с помощью Apache Arrow и Parquet.

REDB  - встраиваемая база данных типа "ключ-значение" на Rust.


Awesome Rust - отобранный список библиотек и ресурсов для Rust.


 

Веб-разработка
Full-stack

Leptos  - веб-фреймворк Rust полного стека, использующий детальную реактивность для создания декларативных пользовательских интерфейсов.



Dioxus  - портативная, производительная и эргономичная среда для создания кроссплатформенных пользовательских интерфейсов на Rust.
 

MoonZoon  - веб-фреймворк Rust полного стека.

 

Front-end

Makepad  - невероятно быстрый кроссплатформенный OSS UI-фреймворк на основе шейдеров с возможностью проектирования в реальном времени. Компилируется в wasm/webGL, osx/metal, windows/dx11 linux/opengl. 

Yew  - современная платформа Rust, похожая на React, предназначенная для создания многопоточных front-end веб-приложений с помощью WebAssembly.
 

Perseus  - полнофункциональный фреймворк, созданный на основе Sycamore.

Trunk  - сборщик веб-приложений WASM для Rust.

Sycamore  - реактивная библиотека для создания веб-приложений на Rust и WebAssembly.

 

Back-end

Rocket  - веб-фреймворк, написанный на Rust, упрощающий написание быстрых, безопасных, гибких и функциональных веб-приложений с невероятным удобством использования.

Actix Web  - мощный, прагматичный и чрезвычайно быстрый веб-фреймворк на Rust.
 

Axum  - веб-фреймворк, ориентированная на эргономику и модульность, который предоставляет основу для создания асинхронных, масштабируемых и поддерживаемых веб-приложений.

Salvo  - чрезвычайно простой и мощный бэкенд-фреймворк. Для разработки бэкенд-сервисов требуются только базовые знания Rust.

Ntex  - мощный, прагматичный и чрезвычайно быстрый фреймворк для компонуемых сетевых (веб) сервисов на Rust.

 

warp  - супер-простой, компонуемый, веб-серверный фреймворк для высоких скоростей.

 

Gotham  - гибкая веб-инфраструктура, созданная для стабильной версии Rust, обеспечивающая стабильность, безопасность, защищённость и скорость.
 

Neon  - библиотека и набор инструментов для встраивания Rust в приложения и библиотеки Node.js.


 

NAPI-RS  - фреймворк для создания предварительно скомпилированных дополнений Node.js на Rust.


 

Настольная и мобильная разработка.
Tauri  - платформа для создания оптимизированных, безопасных и независимых от внешнего интерфейса приложений для всех основных настольных платформ.

Iced  - кроссплатформенная библиотека графического интерфейса для Rust, ориентированная на простоту и безопасность типов.

Slint  - декларативная библиотека для создания нативных пользовательских интерфейсов (GUI) для приложений на языках Rust, C++ или JavaScript.
 

relm4  - идиоматическая библиотека графического интерфейса, вдохновленная Elm и основанная на gtk4-rs для написания нативных приложений.

Rinf  - готовый к использованию фреймворк для создания красивых и производительных кроссплатформенных приложений с использованием Flutter и Rust.

egui  - простая, быстрая и легко переносимая библиотека графического интерфейса, работающая как в браузере, так и в нативной режиме.

Azul  - фреймворк созданный с использованием механизма рендеринга WebRender и CSS/HTML-подобной объектной модели документа для быстрой разработки красивых нативных настольных приложений на Rust, C и C++.

Bevy Engine  - модульный игровой движок, созданный на Rust, с упором на продуктивность и производительность. Бесплатный и с открытым исходным кодом навсегда.



Fyrox Engine  - многофункциональный, готовый к использованию игровой 2D / 3D - движок общего назначения, написанный на Rust с редактором сцен. Ранее известный как rg3d.

Ambient  - игровой движок на Rust и WebAssembly для кроссплатформенных многопользовательских игр.
 

wgpu  - кроссплатформенный, безопасный графический API, написанный на Rust. Подходит для графики как общего назначения, так и вычислений на графическом процессоре.
 

DICOM-rs  - экосистема библиотек и инструментов для работы с системами DICOM.
 

Loro  - высокопроизводительная библиотека на основе бесконфликтных реплицированных типов данных.

 

Отметим, что есть привязки к некоторым популярным инструментам

gtk-rs  - обеспечивает безопасную привязку к языку Rust для фундаментальных библиотек из стека GNOME, таких как GLib, Cairo, GTK 3 и GTK 4.

godot-rust  - проект предоставляющий привязки игрового движка Godot к языку Rust.

 

Инструменты командной строки
r3bl-open-core  - фреймворк для создания программ с терминальными пользовательскими интерфейсоми (TUI).

 

ratatui  - библиотека для создания программ с терминальными пользовательскими интерфейсоми (TUI).

 

clap  - полнофункциональный и быстрый анализатор аргументов командной строки для Rust.

 

command-line interface  - crate's помогающие создавать интерфейсы командной строки.

 

Криптовалюта и блокчейн
Hyperledger  - глобальная экосистема с открытым исходным кодом для технологий блокчейна корпоративного уровня, размещённая Linux Foundation.

awesome-blockchain-rust  - коллекция библиотек и пакетов блокчейна/криптографии для создания собственных блокчейн-приложений на Rust.
 

Искусственный интеллект
Burn  - новая комплексная динамическая среда глубокого обучения, созданная с использованием Rust,
основными целями, которой являются исключительная гибкость, вычислительная эффективность и переносимость.

 

Прочее
Bon  - генератор builder'ов. Создавайте builder'ы для всего.

 

rayon  - библиотека параллелизма данных. Чрезвычайно легкая и позволяет легко преобразовывать последовательные вычисления в параллельные. Также гарантирует отсутствие гонок данных.

Maelstrom  - средство запуска тестов Rust, созданное на базе универсального средства запуска кластерных заданий.
 

teloxide  - полнофункциональный фреймворк, который позволяет легко создавать Telegram ботов с помощью Rust. Сосредоточьтесь только на своей бизнес-логике.
 

CubeCL  - мультиплатформенное расширение языка Rust для высокопроизводительных вычислений.
 

CUDArc  - безопасная обёртка на Rust вокруг инструментария CUDA.
