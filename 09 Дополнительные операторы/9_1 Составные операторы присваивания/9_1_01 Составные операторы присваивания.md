Составные операторы присваивания

В Rust предусмотрены специальные составные (укороченные) операторы присваивания, облегчающие использование некоторых операций присваивания. Рассмотрим пример, когда арифметические операторы применяются к переменной, а полученный результат повторно присваивается ему же:

fn main() {
    let mut num = 0;
    num = num + 3;
    println!("Значение переменной num = {num}"); // "Значение переменной num = 3"
}

В строке num = num + 3; мы берем нашу переменную num и увеличиваем его на 3. Однако писать это каждый раз довольно утомительно. И тут-то на помощь приходят составные операторы присваивания:

fn main() {
    let mut num = 0;
    num += 3; // num = num + 3;
    println!("Значение переменной num = {num}"); // "Значение переменной num = 3"
}

Подобные сокращённые записи есть и для других арифметических и побитовых операций (рассматриваются в следующем уроке). Составные операторы присваивания арифметических операций:

Название	Оператор	Пример    Результат
                        a = -6	

Сложение	+=	        a += 2	  a = -4
Вычитание	-=	        a -= 2	  a = -8
Умножение	*=	        a *= 2	  a = -12
Деление	    /=	        a /= 2	  a = -3
Остаток от
деления	    %=	        a %= 2	  a = 0

Как вы уже могли заметить, их использование заметно удобнее и лаконичнее в определённых случаях.
