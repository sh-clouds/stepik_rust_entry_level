Деление на 2^n
 
Напишите программу, которая считывает два целых числа x (u32) и n (u8), считает деление 
x/2^n, а затем выводит результат.

Тестовые данные ✅

Sample Input:
4
1

Sample Output:
2

Напишите программу. Тестируется через stdin → stdout
Верно решили 83 учащихся
Из всех попыток 93% верных


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {       
    let mut x: u32 = input();
    let mut n: u8 = input();
    println!("{}", x >> n );  
}
