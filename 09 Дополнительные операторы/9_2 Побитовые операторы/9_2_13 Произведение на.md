Произведение на 2^n

Напишите программу, которая считывает два целых числа x (u32) и n (u8), считает произведение 
x∗2^n, а затем выводит результат.

Тестовые данные ✅

Sample Input:
2
1

Sample Output:
4


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {       
    let mut x: u32 = input();
    let mut n: u8 = input();
    println!("{}", x * (1 << n) as u32 );  
}
