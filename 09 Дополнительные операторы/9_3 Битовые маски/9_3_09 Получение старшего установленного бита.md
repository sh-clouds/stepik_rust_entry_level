Получение старшего установленного бита

Напишите программу, которая считывает целое число x (i8), извлекает его старший установленный бит и выводит получившееся число в двоичной и десятичной записи в виде сообщений:

извлеченный из {x} старший установленный бит
в двоичной записи: {:b}
в десятичной записи: {}
Если у числа нет установленных бит, вывести: у числа {:b} нет установленных бит.

Тестовые данные ✅
Sample Input 1:
0

Sample Output 1:
у числа 00000000 нет установленных бит

Sample Input 2:
1

Sample Output 2:
извлеченный из 1 старший установленный бит
в двоичной записи: 00000001
в десятичной записи: 1

Напишите программу. Тестируется через stdin → stdout

use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {       
    let mut x: i8 = input();  
    //x = 3;
    let mut i1: u8 = 0;
    let mut x1: i8 = 0;    
    
    if x == 0
    {
        println!("у числа {:08b} нет установленных бит", x);     
    }
    else
    {
        if x as i16 & 128 != 0 { i1 = 7 }
        else if x & 64 != 0 { i1 = 6 }
        else if x & 32 != 0 { i1 = 5 }
        else if x & 16 != 0 { i1 = 4 } 
        else if x & 8 != 0 { i1 = 3 } 
        else if x & 4 != 0 { i1 = 2 }        
        else if x & 2 != 0 { i1 = 1 }  
        else { i1 = 0}  

        x1 = 1<<i1;
        
        println!("извлеченный из {x} старший установленный бит"); 
        println!("в двоичной записи: {:08b}", x1);     
        println!("в десятичной записи: {}\n", x1);       
    }
}
