Проверка на четность

Напишите программу, которая считывает целое число x (i8), проверяет x на четность и выводит результат:

Если число x является четным, вывести сообщение: число {x} является четным.
Иначе вывести: число {x} является нечетным.
Тестовые данные ✅

Sample Input 1:
0

Sample Output 1:
число 0 является четным

Sample Input 2:
1

Sample Output 2:
число 1 является нечетным

Напишите программу. Тестируется через stdin → stdout

use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {       
    let mut x: i8 = input();     
    //x = 3;
    if x & 1 == 1
    {
        println!("число {x} является нечетным");
    }
    else
    {
        println!("число {x} является четным");    
    }      
}
