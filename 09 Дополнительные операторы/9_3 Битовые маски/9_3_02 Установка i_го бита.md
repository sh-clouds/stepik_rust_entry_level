Установка i-го бита 🪤🪤

Напишите программу, которая считывает два целых числа x (i8) и n (u8), устанавливает n-й бит переменной x и выводит получившееся число в двоичной и десятичной записи в виде сообщений:

x до установки {n}-го бита
в двоичной записи: {:b}
в десятичной записи: {}

x после установки {n}-го бита
в двоичной записи: {:b}
в десятичной записи: {}
Тестовые данные ✅
 
Sample Input:
0
0

Sample Output:
x до установки 0-го бита
в двоичной записи: 00000000
в десятичной записи: 0

x после установки 0-го бита
в двоичной записи: 00000001
в десятичной записи: 1


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {       
    let mut x: i8 = input();
    let mut n: u8 = input();
    println!("x до установки {}-го бита", n); 
    println!("в двоичной записи: {:08b}", x);     
    println!("в десятичной записи: {}\n", x); 

    x |= 1 << n;
    println!("x после установки {}-го бита", n); 
    println!("в двоичной записи: {:08b}", x);     
    println!("в десятичной записи: {}\n", x);     
}
