Позиция старшего установленного бита

Напишите программу, которая считывает целое число x (i8) и выводит позицию (0-7) старшего установленного бита в виде сообщений:

в числе {:b} позиция старшего установленного бита равна {}
Если у числа нет установленных бит, вывести: у числа {:b} нет установленных бит.

Тестовые данные ✅

Sample Input 1:
0

Sample Output 1:
у числа 00000000 нет установленных бит

Sample Input 2:
1

Sample Output 2:
в числе 00000001 позиция старшего установленного бита равна 0

Напишите программу. Тестируется через stdin → stdout

use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {       
    let mut x: i8 = input();  
    //x = 3;
    let mut i1: u8 = 0;   
    
    if x == 0
    {
        println!("у числа {:08b} нет установленных бит", x);     
    }
    else
    {
        if x as i16 & 128 != 0 { i1 = 7 }
        else if x & 64 != 0 { i1 = 6 }
        else if x & 32 != 0 { i1 = 5 }
        else if x & 16 != 0 { i1 = 4 } 
        else if x & 8 != 0 { i1 = 3 } 
        else if x & 4 != 0 { i1 = 2 }        
        else if x & 2 != 0 { i1 = 1 }  
        else { i1 = 0}  
        println!("в числе {:08b} позиция старшего установленного бита равна {}", x, i1);        
    }
}
