Проверка числа на степень 2

Напишите программу, которая считывает целое число x (i8), проверяет x на степень 2 и выводит результат:

Если число x является степенью 2, вывести сообщение: число {x} является степенью 2.
Иначе вывести: число {x} не является степенью 2.
Тестовые данные ✅

Sample Input 1:
0

Sample Output 1:
число 0 не является степенью 2

Sample Input 2:
1

Sample Output 2:
число 1 является степенью 2
Напишите программу. Тестируется через stdin → stdout

Из всех попыток 50% верных



use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {       
    let mut x: i8 = input();     
    //x = 0;
    //let mut x1 = x  & (x - 1);
    if (x>0) && (x  & (x - 1)) == 0
    {
        println!("число {x} является степенью 2");
    }
    else
    {
        println!("число {x} не является степенью 2");    
    }      
}
