Мбит/с
Напишите программу, которая считывает размер файла в байтах (u64) и время передачи файла (f64), а затем выводит (до 2 знаков) скорость передачи данных в Мбит/с по формуле:

Скорость в Мбит/с = Размер файла в байтах×8 / (Время передачи в секундах × 1.000.000)
 
В виде сообщения:

Скорость передачи данных: {} Мбит/с

                  
Тестовые данные ✅

Sample Input:
1024
0.027

Sample Output:
Скорость передачи данных: 0.30 Мбит/с

Напишите программу. Тестируется через stdin → stdout
Верно решили 99 учащихся
Из всех попыток 75% верных


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {
    let n: u64 = input();
    let t: f64 = input();
    println!("Скорость передачи данных: {:.2} Мбит/с", (n as f64) * 8.0 / t / 1000000.0);
}
