Площадь круга

Напишите программу, которая считывает радиус (f64) и выводит (до 3 знаков) площадь круга. Используйте константу для значения π=3.141592653589793.

Тестовые данные ✅

Sample Input:
4.5

Sample Output:
63.617

Напишите программу. Тестируется через stdin → stdout
Верно решили 95 учащихся
Из всех попыток 90% верных



use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {
    const pi: f64 = 3.141592653589793;
    let r: f64 = input();
    println!("{:.3}", pi * r * r);
}
