Напишите программу, которая считывает два целых числа и меняет их значения местами без использования третьей переменной.

Так, если переменные a = 5, b = 10, то после обмена значениями должно стать a = 10, а b = 5 без использования дополнительной переменной.

Подсказка 💡
Для решения данной задачи нужно от первого числа отнять второе число и присвоить результат первому.
Ко второму числу добавить новое значение первого числа.
Затем от второго отнять новое значение первого и присвоить первому. 

Тестовые данные ✅

Sample Input:
2
3

Sample Output:
3
2

Напишите программу. Тестируется через stdin → stdout

use std::io;

fn main() {
    // put your Rust code here
    let mut s1 = String::new();
    let mut s2 = String::new();    
    
    io::stdin()
        .read_line(&mut s1) 
        .expect("Не удалось прочитать ввод"); 

    io::stdin()
        .read_line(&mut s2) 
        .expect("Не удалось прочитать ввод");   
    
    let mut num1: i32;
    let mut num2: i32;    
    num1 = s1
        .trim()
        .parse() 
        .expect("Пожалуйста, введите число!");
    
    num2 = s2
        .trim()
        .parse() 
        .expect("Пожалуйста, введите число!");    
    
    num1 = num1 - num2;
    num2 += num1;
    num1 = num2 - num1;
    println!("{}\n{}", num1, num2);
}
