Восток дело тонкое

Далеко на востоке, на границе с соседним королевством стоял величественный замок, охраняемый мудрым старцем-хранителем. Этот старец был известен своей скромностью и умением находить решения для самых сложных проблем. Однажды в замок пришли послы из соседнего королевства, предлагая заключить договор о сотрудничестве. Они предложили провести совместное исследование на тему долгосрочного увеличения урожайности на землях обоих королевств. Старец согласился, и началась совместная работа.

Старец понял, что для успешного исследования необходимо рассчитать остаток от деления числа площади поля на число использованных для посадки растений. Это позволило бы определить оптимальное распределение ресурсов для увеличения урожайности. Ваша задача помочь старцу выполнить это вычисление.

Напишите программу, которая считывает два целых числа и выводит остаток от деления первого числа на второе.

Тестовые данные ✅

Sample Input:
10
1

Sample Output:
На поле доступно еще 0 кв.м свободного места

Напишите программу. Тестируется через stdin → stdout
Верно решил 431 учащийся

use std::io;

fn main() {
    // put your Rust code here
    let mut s1 = String::new();
    let mut s2 = String::new();    
    
    io::stdin()
        .read_line(&mut s1) 
        .expect("Не удалось прочитать ввод"); 

    io::stdin()
        .read_line(&mut s2) 
        .expect("Не удалось прочитать ввод");   
    
    let num1: i32;
    let num2: i32;    
    num1 = s1
        .trim()
        .parse() 
        .expect("Пожалуйста, введите число!");
    
    num2 = s2
        .trim()
        .parse() 
        .expect("Пожалуйста, введите число!");    
  
    println!("На поле доступно еще {} кв.м свободного места", num1 % num2);
}
