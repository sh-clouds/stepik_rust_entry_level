Арифметика систем

Напишите программу, которая считывает два целых числа с консоли в десятичной системе и выводит в двоичной, восьмеричной, шестнадцатеричной системах арифметические операции (+, -, *, /, %) над первым числом.

Уточним, что каждая арифметическая операция должна быть выведена в каждой из систем счисления. При выводе операций для следующей системы счисления используйте переход на новую строку. Для ясности посмотрите на входные и выходные данные. Гарантируется, что второе число не равно 0!

Тестовые данные ✅
Sample Input:
5
2

Sample Output:
0b101 + 0b10 = 0b111
0o5 + 0o2 = 0o7
0x5 + 0x2 = 0x7

0b101 - 0b10 = 0b11
0o5 - 0o2 = 0o3
0x5 - 0x2 = 0x3

0b101 * 0b10 = 0b1010
0o5 * 0o2 = 0o12
0x5 * 0x2 = 0xa

0b101 / 0b10 = 0b10
0o5 / 0o2 = 0o2
0x5 / 0x2 = 0x2

0b101 % 0b10 = 0b1
0o5 % 0o2 = 0o1
0x5 % 0x2 = 0x1

Напишите программу. Тестируется через stdin → stdout

use std::io;

fn main() {
    // put your Rust code here
    let mut s1 = String::new();
    let mut s2 = String::new();    
    
    io::stdin()
        .read_line(&mut s1) 
        .expect("Не удалось прочитать ввод"); 

    io::stdin()
        .read_line(&mut s2) 
        .expect("Не удалось прочитать ввод");   
    
    let num1: i32;
    let num2: i32;    
    num1 = s1
        .trim()
        .parse() 
        .expect("Пожалуйста, введите число!");
    
    num2 = s2
        .trim()
        .parse() 
        .expect("Пожалуйста, введите число!");    
  
    println!("{:#b} + {:#b} = {:#b}", num1, num2, num1 + num2);
    println!("{:#o} + {:#o} = {:#o}", num1, num2, num1 + num2);    
    println!("{:#x} + {:#x} = {:#x}\n", num1, num2, num1 + num2);  
    
    println!("{:#b} - {:#b} = {:#b}", num1, num2, num1 - num2);
    println!("{:#o} - {:#o} = {:#o}", num1, num2, num1 - num2);    
    println!("{:#x} - {:#x} = {:#x}\n", num1, num2, num1 - num2); 
    
    println!("{:#b} * {:#b} = {:#b}", num1, num2, num1 * num2);
    println!("{:#o} * {:#o} = {:#o}", num1, num2, num1 * num2);    
    println!("{:#x} * {:#x} = {:#x}\n", num1, num2, num1 * num2);     
    
    println!("{:#b} / {:#b} = {:#b}", num1, num2, num1 / num2);   
    println!("{:#o} / {:#o} = {:#o}", num1, num2, num1 / num2);    
    println!("{:#x} / {:#x} = {:#x}\n", num1, num2, num1 / num2);    
    
    println!("{:#b} % {:#b} = {:#b}", num1, num2, num1 % num2);
    println!("{:#o} % {:#o} = {:#o}", num1, num2, num1 % num2);    
    println!("{:#x} % {:#x} = {:#x}", num1, num2, num1 % num2);
}
