Изменяемость

В Rust переменные по умолчанию неизменяемы. Как только мы инициализировали переменную, мы не можем её изменить:

fn main() {
    let age = 25;
    age = 23; // ошибка, невозможно изменить неизменяемую переменную
}
Для того чтобы сделать переменную изменяемой, необходимо указать ключевое слово mut перед именем переменной:

fn main() {
    let mut age = 25;
    age = 23; // теперь в age хранится 23
}
Для сторонних разработчиков указание вами переменной как mut обозначает возможность изменения её значения другими частями кода. Также значение одной переменной можно присвоить другой при условии, что они одинакового типа.

fn main() {
    let x = 2;
    let y = x; // y равен 2
}
Привязки
В действительности левая часть инструкции let представляет собой шаблон, который связывает правое значение с именем переменной, образуя привязки (bindings) переменных:

fn main() {
    let x = 2; // x - левая часть, 2 - правая часть
}
Благодаря этому мы можем определять сразу несколько переменных:

fn main() {
    let (x, y) = (1, 2); // x = 1, y = 2
    let (mut a, mut b) = (3, 4); // a = 3, b = 4
}
В отличие от других языков, привязки в Rust имеют ещё один аспект. Они должны быть инициализированы, прежде чем будут использованы:

fn main() {
    let x: i32;
    println!("The value of x is: {}", x); // ошибка
}
Тема шаблонов выходит за рамки данного курса, поэтому этого материала пока достаточно.
