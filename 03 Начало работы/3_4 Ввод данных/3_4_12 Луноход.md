Луноход

Космический луноход "Исследователь 1" отправлен на Луну для изучения новых минеральных образцов. Однако в ходе своей миссии он столкнулся с серьезной неисправностью и перестал передавать данные на Землю. Перед полным выходом из строя он смог отправить последнее сообщение, но из-за неисправности оно было отправлено задом наперед. Теперь команда на Земле должна разгадать это сообщение и понять, какие минералы были обнаружены.

Напишите программу, которая считывает сообщение (четыре строки) отправленное луноходом в обратном порядке, и выводит его в правильном порядке (в одну строку), чтобы команда на Земле могла понять, какие минералы были обнаружены.

Тестовые данные ✅

Sample Input:
латерит
минерал
новый
Обнаружен

Sample Output:
Обнаружен новый минерал латерит

Напишите программу. Тестируется через stdin → stdout
Верно решили 537 учащихся
Из всех попыток 60% верных


use std::io;

fn main() {
    // put your Rust code here
    let mut user_input = String::new();

    io::stdin()
        .read_line(&mut user_input) 
        .expect("Не удалось прочитать ввод"); 
    
    io::stdin()    
        .read_line(&mut user_input)  
        .expect("Не удалось прочитать ввод"); 
    
    io::stdin()      
        .read_line(&mut user_input)  
        .expect("Не удалось прочитать ввод");  
    
    io::stdin()      
        .read_line(&mut user_input)  
        .expect("Не удалось прочитать ввод");      
    
    let out = user_input.split("\n")
		.collect::<Vec<&str>>()    
 		.into_iter()   
		.rev()    
		.collect::<Vec<&str>>()    
		.join(" ");    
    
    println!("{}", out);
}
