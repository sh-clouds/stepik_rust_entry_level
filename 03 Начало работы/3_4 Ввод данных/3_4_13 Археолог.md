Археолог

Вы археолог, исследующий древние культуры. В ходе раскопок вы обнаружили древний свиток с записями, написанными на древнем языке. Однако, чтобы расшифровать записи, вам нужно преобразовать числа, записанные на свитке, из строкового формата в обычное целое число. Только тогда вы сможете понять тайны древних знаний.

Напишите программу, которая считывает строковое число с консоли, преобразует его в обычное целое число и затем выводит на консоль.

1

Sample Output 1:
1

Sample Input 2:
10

10

use std::io;

fn main() {
    // put your Rust code here
    let mut ui = String::new();
    
    io::stdin()
        .read_line(&mut ui) 
        .expect("Не удалось прочитать ввод"); 
      
    let num: i32;
    num = ui
        .trim()
        .parse() 
        .expect("Пожалуйста, введите число!");
  
    println!("{num}");
}
