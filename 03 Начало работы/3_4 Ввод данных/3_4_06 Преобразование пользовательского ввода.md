Преобразование пользовательского ввода

Введённые пользователем данные всегда представлены в виде строки, и неважно, ввёл он текст или число. В предыдущем примере, когда мы написали let mut user_input = String::new() компилятор определил, что user_input строкового типа, а значит, может хранить только строку. Так, ввод числа 12 выглядит как "12" а не 12. Возможно, некоторые предложат сделать user_input числовым типом, скажем, i32, однако это не сработает. Дело в том, что метод read_line() принимает переменную только типа String, попытка передать переменную числового типа вызовет ошибку:

use std::io;

fn main() {
    let mut user_input: i32;

    // ошибка
    io::stdin()
        .read_line(&mut user_input)
        .expect("Не удалось прочитать ввод");
}

Как же тогда нам считать число? В этом случае нам необходимо преобразовать введённую пользователем строку в нужный нам числовой тип. При этом для переменной, куда мы хотим сохранить преобразование, требуется аннотация типа, что логично, поскольку компилятор должен знать его тип во время компиляции. Приступим:

Считаем пользовательский ввод:
use std::io;

fn main() {
    let mut user_input = String::new();

    io::stdin()
        .read_line(&mut user_input)
        .expect("Не удалось прочитать ввод");
}

Определим новую переменную in_num для сохранения преобразования(числа):

use std::io;

fn main() {
    let mut user_input = String::new();

    io::stdin()
        .read_line(&mut user_input)
        .expect("Не удалось прочитать ввод");

    let in_num: i32 =
}

Перед конвертацией строки в число необходимо убедиться, что в строке отсутствуют пробелы и символы новой строки. К примеру, попытка преобразовать " 12 " вызовет ошибку. Для этого у переменных типа String существует метод trim() который возвращает новую строку, удаляя любые пробельные символы (включая \n) в начале и в конце строки, если они есть. Вызовем trim() у user_input:

use std::io;

fn main() {
    let mut user_input = String::new();

    io::stdin()
        .read_line(&mut user_input)
        .expect("Не удалось прочитать ввод");

    let in_num: i32 = user_input.trim()
}

Теперь, когда мы предусмотрели наличие пробельных символов, можно преобразовать строку в число с помощью метода parse() который преобразует строку в другой тип:
use std::io;

fn main() {
    let mut user_input = String::new();

    io::stdin()
        .read_line(&mut user_input)
        .expect("Не удалось прочитать ввод");

    let in_num: i32 = user_input.trim().parse();
}

Последний шаг состоит в обработке случая, когда пользователь, помимо числа, ввёл другие символы, например, "2%". Вызовем метод expect() у перечисления Result, которую возвращает parse(). В случае успешного преобразования строки expect() вернёт число, в противном - аварийно завершит программу и отобразит переданное ему сообщение, если оно было передано.

use std::io;

fn main() {
    let mut user_input = String::new();

    io::stdin()
        .read_line(&mut user_input)
        .expect("Не удалось прочитать ввод");

    let in_num: i32 = user_input
        .trim()
        .parse()
        .expect("Пожалуйста, введите число!");
}

Использование expect() является обязательным. Без него программа не сработает.
Добавим ещё вывод значения in_num, прибавив к нему 1:

use std::io;

fn main() {
    println!("Пожалуйста, введите ваше значение");

    let mut user_input = String::new();

    io::stdin()
        .read_line(&mut user_input)
        .expect("Не удалось прочитать ввод");

    let in_num: i32 = user_input
        .trim()
        .parse()
        .expect("Пожалуйста, введите число!");

    println!("{in_num} + 1 = {}", in_num + 1);
}
