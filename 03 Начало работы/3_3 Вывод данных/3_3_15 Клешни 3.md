В редакторе кода представлен фрагмент кода. Необходимо:

1. Дополнить println!() в пятой строке аргументами a, b, c, d, e, f в правильном порядке, чтобы получить корректный вывод. Другими словами, требуется решить задачу, не изменяя представленные индексы.

Обратите внимание на комментарий // "a = 5, b = 10, c = 15, d = 20, e = 25, f = 30"

2. Вывести через второй println!() в шестой строке получившуюся последовательность аргументов. Так, если у вас получилось println!("a = {3}, b = {}, c = {5}, d = {}, e = {4}, f = {}", a, b, c, d, e, f), то в шестой строке будет println!("a, b, c, d, e, f").
