Римские цифры

В редакторе кода представлена неполная программа, которую необходимо доделать. Программа должна считать год (u16) и вывести его римский эквивалент, используя:

I = 1	XL = 40	D = 500
IV = 4	L = 50	CM = 900
V = 5	XC = 90	M = 1000
IX = 9	C = 100	 
X = 10	CD = 400	

Вывод и вычисление реализуйте в функции print_rome() ! Гарантируется, что год задан корректно !

Тестовые данные ✅

Sample Input:
1

Sample Output:
I

Напишите программу. Тестируется через stdin → stdout

Из всех попыток 82% верных


use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {
    let mut year = input();         
    print_rome(year);
}

fn print_rome(mut year: u16) {
    let values = [1000, 900, 500, 400, 100, 90, 50, 40, 10, 9, 5, 4, 1];
    let symbols = ["M", "CM", "D", "CD", "C", "XC", "L", "XL", "X", "IX", "V", "IV", "I"];
    let mut roman_num = String::new();
    let mut index = 0;

    while year > 0 {
        for _ in 0..((year as i32) / values[index]) {
            roman_num.push_str(symbols[index]); // Добавляем римскую цифру
            year -= (values[index] as u16); // Вычитаем значение
        }
        index += 1;
    }
    println!("{roman_num}");    
}
