Количество вариантов

В редакторе кода представлена неполная программа, которую необходимо доделать. Программа должна считать размерность алфавита (u8) и длину слова (u8), а затем вывести количество слов, которые можно составить из заданного алфавита. 

Нахождение количества слов реализуйте в функции word_count() ! Результирующее значение должно быть типа u64. 

Тестовые данные ✅
 
Sample Input:
2
5

Sample Output:
32

Напишите программу. Тестируется через stdin → stdout

use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {
    let word_len: u8 = input();
    let alph_len: u8 = input();   
    let res: u64 = word_count(word_len, alph_len);        
    println!("{res}");
}

fn word_count(word_len: u8, alph_len: u8) -> u64 {
    (word_len as u64).pow(alph_len as u32) as u64
}
