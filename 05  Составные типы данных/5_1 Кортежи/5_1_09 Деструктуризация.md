Деструктуризация

Значения кортежа могут быть извлечены в отдельные переменные. Данный процесс именуется деструктуризацией (destructuring):

fn main() {
    let tup = (1, 2.0, '3', "Hello", true);

    let (i_num, f_num, ch, str_item,b) = tup; // деструктуризация

    println!("i_num = {i_num}, f_num = {f_num}, ch = {ch}, str_item = {str_item}, b = {b}");

    // "i_num = 1, f_num = 2, ch = 3, str_item = Hello, b = true"
}

Пустой кортеж

Пустой кортеж, который не имеет значений, обладает единичным типом (unit type). Тип и значение пустого кортежа записываются пустыми скобками (). 

fn main() {
    let em_tup = (); // пустой кортеж
    //Или
    let em_tup: () = (); // пустой кортеж
    //Или
    let em_tup: (); // пустой кортеж
}

Единичный тип применяется при возврате пустых значений из других выражений. Случаи его использования станут более понятны по мере изучения языка.
