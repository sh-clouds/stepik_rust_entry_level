Чашечку кофе ☕

При покупке каждых 8 чашек кофе покупатель получает ещё одну бесплатно.

В редакторе кода представлена неполная программа, которую необходимо доделать. Программа должна считать количество покупаемых чашек кофе (u32) и цену за одну чашку (f64), а затем вывести (до 2 знаков) сумму к оплате.

Нахождение итоговой суммы реализуйте в функции get_сoffee_сost() !

Тестовые данные ✅

Sample Input:
1
350.5

Sample Output:
350.50

Напишите программу. Тестируется через stdin → stdout

use std::io;
use std::str::FromStr;

fn input<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Failed to read input");
    input.trim().parse::<T>().expect("Failed to parse input")
}

fn main() {
    let mut cups: u32 = input();
    let mut cup_cost: f64 = input();    
    let mut coffee_cost = get_coffee_cost(cups, cup_cost);   
    println!("{coffee_cost:.2}");  
}


fn get_coffee_cost(cups: u32, cup_cost: f64) -> f64 {
   let mut free_cups = cups / 9;
   return  (cups - free_cups) as f64 * cup_cost;   
}
