iSQRT

Реализация функции для вычисления квадратного корня может быть выполнена несколькими способами. Один из самых известных методов - это метод Ньютона (или метод касательных). Метод Ньютона основан на итеративной формуле:

x_n+1 = 1/2 * (x_n + S/x_n),

где S - число, квадратный корень которого мы ищем, а 
x_n - текущее приближение.

В редакторе кода представлена неполная программа, которую необходимо доделать. Программа должна считывать неотрицательное целое число (u32) и выводить (до 11 знаков) квадратный корень данного числа с точностью 1e-10. 

Вычисление квадратного корня реализуйте в функции sqrt() !

fn sqrt(mut num: u32) -> f64 {
    let accur = 1e-10;
    if num == 0 {
        return 0.0;
    }
    let mut x = num as f64;
    loop {
        let mut next_x = 0.5 * (x + (num as f64) / x);
        if abs(next_x - x) < accur {
            return next_x;
        }
        x = next_x;
    }
}

fn main() {
    let mut input = String::new();
    std::io::stdin()
        .read_line(&mut input)
        .expect("Не удалось прочитать строку");

    let num: u32 = input.trim().parse().expect("Пожалуйста, введите натуральное число");

    println!("{:.11}", sqrt(num));
}
fn abs(num: f64) -> f64 {
    if num < 0.0 {
        return -num;
    } else {
        return num;
    }
}
