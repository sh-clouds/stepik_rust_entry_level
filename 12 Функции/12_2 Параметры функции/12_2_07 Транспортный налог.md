Транспортный налог 

Транспортный налог в РФ - это региональный налог. Им облагаются не только автомобили, но и другие виды транспорта, например яхты, катера, самолеты, вертолеты и др.

В редакторе кода представлена неполная программа, которую необходимо доделать. Программа должна считывать налоговую ставку в рублях (f64), мощность транспорта в л.с (u16) и период владения в месяцах (u8), а затем выводить (до 2 знаков) транспортный налог по следующей формуле:

транспортный налог = ставка налога × мощность × (период владения / 12)

Для считывания значений используйте функцию input(), расчёт транспортного налога произвести в calc_tax(), а вывод в print_tax().

Тестовые данные ✅

Sample Input:
12
98
12

Sample Output:
1176.00

use std::io;
use std::str::FromStr;

fn input1<T: FromStr>() -> T 
where
    T::Err: std::fmt::Debug,
{
    let mut input1 = String::new();
    std::io::stdin()
        .read_line(&mut input1)
        .expect("Failed to read input");
    input1.trim().parse::<T>().expect("Failed to parse input")
}

fn input() {
    let tax_rate: f64  = input1();
    let veh_pow: u16 = input1();
    let own_per: u8 = input1();        
    calc_tax(tax_rate, veh_pow, own_per);
}

fn main() {
    input();
}

fn calc_tax(tax_rate: f64, veh_pow: u16, own_per: u8) {    
    print_tax(tax_rate * veh_pow as f64  * own_per as f64 / 12.0);
}

fn print_tax(tax: f64) {
    println!("{tax:.2}")
}
