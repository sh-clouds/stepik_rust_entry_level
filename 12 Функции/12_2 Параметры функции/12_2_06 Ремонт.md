Ремонт 

Строительство любого дома включает в себя укладку стяжки пола, которая рассчитывается умножением площади на цену одного квадратного метра.

В редакторе кода представлена неполная программа, которую необходимо доделать. Программа должна считывать длину (f64) и ширину (f64) пола, и стоимость (f64) одного квадратного метра, а затем вывести (до 2 знаков) стоимость стяжки.

Для считывания сторон используйте функцию get_len_width(). Расчёт стоимости произвести в calc_cost(), а вывод в print_cost().

Тестовые данные ✅

Sample Input:
1
1
5

Sample Output:
5.00

Напишите программу. Тестируется через stdin → stdout

use std::io;

fn get_len_width() {
    let mut input = String::new();
    io::stdin().read_line(&mut input).unwrap();
    let len = input.trim().parse::<f64>().unwrap();  
    
    input.clear();
    let mut input = String::new();
    io::stdin().read_line(&mut input).unwrap();
    let width = input.trim().parse::<f64>().unwrap(); 
    
    input.clear();
    let mut input = String::new();
    io::stdin().read_line(&mut input).unwrap();
    let price = input.trim().parse::<f64>().unwrap(); 
    
    calc_cost(len, width, price);
}

fn main() {
    get_len_width();
}

fn calc_cost(length: f64, width: f64, price: f64) {
    let mut cost = length * width * price;
    print_cost(cost);
}

fn print_cost(cost: f64) {
    println!("{cost:.2}");
}
