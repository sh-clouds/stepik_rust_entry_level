Выражения

По возможности используйте ориентированную на выражения природу Rust:

// ok
let x = if y { 1 } else { 0 };
// nope
let x;
if y {
    x = 1;
} else {
    x = 0;
}

              

Именование

Типы должны быть UpperCamelCase,
Варианты перечисления должны быть UpperCamelCase,
Поля структур должны быть snake_case,
Имена функций и методов должны snake_case,
Локальные переменные должны быть snake_case,
Имена макросов должны быть snake_case,
Константы (const и immutable static) должны быть SCREAMING_SNAKE_CASE.
Когда имя недопустимо, потому что это зарезервированное слово (например, crate), используйте либо сырой (необработанный) идентификатор (r#crate), либо символ нижнего подчеркивания в конце (crate_). Не допускайте ошибок в написании слова (krate).
